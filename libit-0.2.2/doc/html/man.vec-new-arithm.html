vec_new_arithm

Name

vec_new_arithm -- create an arithmetic sequence

Synopsis

#include <it/vec.h>

vec vec_new_arithm( double start, double incr, size_t N );

DESCRIPTION

The function vec_new_arithm creates a vector generated so that the elements correspond to the arithmetic sequence of starting value start, increasing by a step of incr and of length N.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_arithm(5,-2.5,4); /* creates the vector [5 2.5 0 -2.5] */
vec_delete(v);                     /* free the vector                */
