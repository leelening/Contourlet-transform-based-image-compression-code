ivec_new_union

Name

ivec_new_union -- Merge two vectors

Synopsis

#include <it/vec.h>

ivec ivec_new_union( ivec v1, ivec v2 );

DESCRIPTION

Note that this function is different from the function bvec_concat in so far as it considers the vectors as sets of distinct values, and that the vector returned is composed of distinct values only.
