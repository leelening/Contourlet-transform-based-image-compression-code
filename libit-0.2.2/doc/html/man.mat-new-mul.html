mat_new_mul

Name

mat_new_mul -- Performs matrix multiplication

Synopsis

#include <it/mat.h>

mat mat_new_mul( mat m1, mat m2 );

DESCRIPTION

The function mat_new_mul creates a new matrix containing the result of m1xm2. It performs to dimension checking.

EXAMPLE

#include <mat.h>

...

mat m1 = mat_new (2,2);
mat m2 = mat_new (2,2);
mat m3; 

m1[0][0] = 1.; m1[0][1] = 3.; 
m1[1][0] = 2.; m1[1][1] = 4.; 

m2[0][0] = 1.; m2[0][1] = 3.; 
m2[1][0] = 2.; m2[1][1] = 4.; 

m3 = mat_new_mul (m1, m2); 

/*
m3 = 
[[ 7   15 ]
 [ 10  20 ]]
*/
