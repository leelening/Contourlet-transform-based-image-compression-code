vec_find_sorted

Name

vec_find_sorted -- find the position of a value in a sorted vector

Synopsis

#include <it/vec.h>

idx_t vec_find_sorted( vec v, double val );

DESCRIPTION

Assuming that the vector is sorted, the function vec_find_sorted returns a position in the vector v where the value val can be found. If the value val is not in the vector, the function returns the position where is should be inserted to maintain the vector sorted. In particular, is the value val is greater than any element of the vector, the function returns the length of the vector.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_arithm (1,1, 4);      /* v =[1,2,3,4]   */

idx_t pos = vec_find_sorted (v, 3);   /* pos = 2 */          
pos = vec_find_sorted (v, 2.2);       /* pos = 2 */    
pos = vec_find_sorted (v, -1);        /* pos = 0 */    
pos = vec_find_sorted (v, 5);         /* pos = 4 */
