ivec_cum_sum

Name

ivec_cum_sum -- cumulative sum of a vector

Synopsis

#include <it/vec.h>

ivec ivec_cum_sum( ivec v );

DESCRIPTION

The function ivec_cum_sum returns the cumulative sum of the vector v. The cumulative sum is a vector defined so that the component i of this vector represent the summation of the element of v from position 0 to position i (included).

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string( "5 -2 2 4 1" );  /* Create a vector of 5 elements */
ivec s = ivec_cum_sum( v );                /* s = [5 3 5 9 10]              */
