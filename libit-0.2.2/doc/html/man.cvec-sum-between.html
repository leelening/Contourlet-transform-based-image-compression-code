cvec_sum_between

Name

cvec_sum_between -- partial sum of a vector

Synopsis

#include <it/vec.h>

cplx cvec_sum_between( cvec v, size_t i1, size_t i2 );

DESCRIPTION

The function cvec_sum_between returns the sum of the elements of the vector v whose position stands in the interval [i1,i2]. The keyword end may be used for the parameter i2. In that case it means that the summation must be proceeded from position i1 to the end of the vector.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string( "i 1-2i 2.5 0 -i" ); /* Create a vector of 5 elements */
cplx s = cvec_sum_between( v, 2, end );        /* s = 2.5-i                     */
