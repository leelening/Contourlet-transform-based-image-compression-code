imat_col_decr

Name

imat_col_decr -- substract a value to elements of a given column

Synopsis

#include <it/mat.h>

imat imat_col_decr( imat m, size_t col, int a );

DESCRIPTION

The function imat_col_decr substracts a to all the elements of the column col in the matrix m
