cvec_new_reverse

Name

cvec_new_reverse -- Reverse a vector

Synopsis

#include <it/vec.h>

cvec cvec_new_reverse( cvec v );

DESCRIPTION

This function creates a new vector composed of the element of the input vector v taken in the backward order.

EXAMPLE

#include <vec.h>

...

cvec v1 = cvec_new_string("[1 2 3 4]");
cvec v2 = cvec_new_reverse(v1);        /* The vector [4 3 2 1] */
