ivec_reverse

Name

ivec_reverse -- Reverse a vector

Synopsis

#include <it/vec.h>

void ivec_reverse( ivec v );

DESCRIPTION

The function ivec_reverse reverse the elements of the vector v.

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string("1 2 3 4");       /* Vector [1 2 3 4] */
ivec_reverse(v);                           /* v = [4 3 2 1]    */
