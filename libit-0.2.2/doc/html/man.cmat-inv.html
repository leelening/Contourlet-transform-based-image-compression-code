cmat_inv

Name

cmat_inv -- computes inverse of complex matrix

Synopsis

#include <it/linalg.h>

cmat cmat_inv( cmat m );

DESCRIPTION

The function cmat_inv is the complex-analogon for mat_inv, except that it: - returns a void matrix instead of NULL when m is singular - does not use complex LU decomposition
