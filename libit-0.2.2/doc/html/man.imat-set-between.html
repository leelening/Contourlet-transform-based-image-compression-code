imat_set_between

Name

imat_set_between -- Set some matrix components to a value

Synopsis

#include <it/mat.h>

imat imat_set_between( imat m, size_t r1, size_t c1, size_t r2, size_t c2, int val );

DESCRIPTION

The imat_set_between function sets all the elements of the matrix m whose position stands in the index interval [r1,r2]x[c1,c2] to the value val. See also: vec_set_between

EXAMPLE

#include <mat.h>

...

See <function>mat_set_between</function>, <function>vec_set_between</function>
