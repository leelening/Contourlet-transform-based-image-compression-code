vec_log10

Name

vec_log10 -- logarithm base 10 to the components of a vector

Synopsis

#include <it/vec.h>

void vec_log10( vec v );

DESCRIPTION

The function vec_exp applies the function logarithm to all the components of v. The base of the logarithm is 10.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string("[1 10 100 1000]");   /* vector [1 10 100 1000] */
vec_log( v );                                /* v = [0 1 2 3]          */
