mt19937_rand_int31

Name

mt19937_rand_int31 -- generates an uniform pseudo-random number (31 bit)

Synopsis

#include <it/random.h>

int mt19937_rand_int31( void );

DESCRIPTION

This function generates an uniformly-distributed pseudo-random number in the [0,0x7fffffff]-interval. Like all other functions having mt19937 in their name, they are intended to use advanced features of the MT19937.

EXAMPLE

#include <random.h>

...

mt19937_srand( 12345 );          /* init with seed 12345               */
int r = mt19937_rand_int31();    /* r is the first int from above seed */
