mat_solve_mat

Name

mat_solve_mat -- outputs X such that AX=B

Synopsis

#include <it/linalg.h>

mat mat_solve_mat( mat A, mat B );

DESCRIPTION

This function uses LU decomposition to repeatedly solve A*X_i = X_i. The b_i are the columns of matrix B. The first step of computing LU decomposition is O(n^3), but solving every equation takes O(n^2). The matrix A must be square. See mat_solve_vec.

EXAMPLE

#include <linalg.h>

...

mat a = mat_new( 2, 2 ); 
mat b = mat_new( 2, 2 );
vec b = vec_new( 2 ); 

a[0][0] = 1; a[0][1] = 3; 
a[1][0] = 5; a[1][1] = 3; 

b[0][0] = 3; b[0][1] = 1; 
b[1][0] = 1; b[1][1] = 3; 

mat x = mat_solve_mat( a, b ); /* x = [-.5 .5 ; 1.666666667 .166666667] */
