mat_height_max

Name

mat_height_max -- Get current maximum height of input matrix

Synopsis

#include <it/mat.h>

int mat_height_max( mat m );

DESCRIPTION

The function mat_height_max returns the maximum height of a matrix. By maximum height, it means the height to which the matrix can be set without calling a free/malloc function.

EXAMPLE

#include <mat.h>

...

mat m = mat_new (123,123);

int hmax = mat_height_max (m);  /* hmax = 123 */
