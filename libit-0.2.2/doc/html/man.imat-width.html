imat_width

Name

imat_width -- get the width of a matrix

Synopsis

#include <it/mat.h>

int imat_width( imat m );

DESCRIPTION

This macro returns the width of the matrix m.

EXAMPLE

#include <mat.h>

...

imat m = imat_new(5, 7);
int w  = imat_width(m); /* w = 7 */
