mat_cholesky

Name

mat_cholesky -- performs in-place Cholesky decomposition

Synopsis

#include <it/linalg.h>

void mat_cholesky( mat A );

DESCRIPTION

The function mat_cholesky performs in-place Cholesky decomposition of a symmetric positive-definite (spd) matrix. We do not check whether A is spd or not. Cholesky decomposition is similar to LU but returns a lower triangular matrix L such that A=LL^t.

EXAMPLE

#include <linalg.h>

...

mat a= mat_new (2,2); 

a[0][0] = 12.;   a[0][1] = 2.; 
a[1][0] = 2.;   a[1][1] = 11.; 

mat_cholesky( a ); /* a=[3.464 0.000 ; 0.577 3.266] */
