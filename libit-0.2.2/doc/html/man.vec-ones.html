vec_ones

Name

vec_ones -- set all the elements of a vector to one

Synopsis

#include <it/vec.h>

void vec_ones( vec v );

DESCRIPTION

The function vec_ones sets all the elements of an already existing vector to 1.

EXAMPLE

#include <vec.h>

...

vec v = vec_new(5); /* creates a vector of size 5 (not initialized) */
vec_ones(v);        /* the vector is set to [1 1 1 1 1]             */
vec_delete(v);      /* free the vector                              */
