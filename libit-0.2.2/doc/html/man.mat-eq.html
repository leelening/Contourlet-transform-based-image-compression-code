mat_eq

Name

mat_eq -- Test matrices for equality

Synopsis

#include <it/mat.h>

int mat_eq( mat m1, mat m2 );

DESCRIPTION

The function mat_eq tests if m1 and m2 are equal and return a non-zero integer if m1==m2, or zero otherwise.

EXAMPLE

#include <mat.h>

...

mat m1, m2;
m1 = mat_new_zeros(2,2);
m2 = mat_new_ones(2,2);
if(mat_eq(m1, m2))
  printf("the matrices are equal");
else
  printf("the matrices are not equal");
