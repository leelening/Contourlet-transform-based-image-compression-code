mat_det

Name

mat_det -- computes the determinant of the input matrix

Synopsis

#include <it/linalg.h>

double mat_det( mat a );

DESCRIPTION

The function mat_det computes the determinant of the matrix a using LU decomposition. Singular matrices have determinant equal to zero.

EXAMPLE

#include <linalg.h>

...

mat m1 = mat_new( 3, 3 ); 
double d; 

m1[0][0] = 1;   m1[0][1] = 5;   m1[0][2] = 2; 
m1[1][0] = 2;   m1[1][1] = 3;   m1[1][2] = 5; 
m1[2][0] = 3;   m1[2][1] = 7;   m1[2][2] = 3; 

d = mat_det (m1);        /* d = 29.000000 */
