ivec_set_length

Name

ivec_set_length -- Modify the length of a vector

Synopsis

#include <it/vec.h>

void ivec_set_length( ivec v, size_t N );

DESCRIPTION

The ivec_set_length macro modifies the length of the vector v so that it is equal to N. If the length increases, the additional elements are not initialized. If the amount of memory allocated to the vector is not sufficient, then the function ivec_set_length_max is automatically called to allow the size modification. For this reason, the pointer corresponding to the vector may be modified.

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new(10);  /* allocate a new vector of size 10 */
ivec_set_length(v,4);   /* reduce the size of the vector    */
ivec_set_length(v,12);  /* increases the size of the vector */
v[10] = 1;
v[11] = 4;
ivec_delete(v);                        /* free the vector */
