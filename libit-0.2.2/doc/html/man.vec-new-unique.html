vec_new_unique

Name

vec_new_unique -- set of distinct values of a vector

Synopsis

#include <it/vec.h>

vec vec_new_unique( vec v );

DESCRIPTION

This function returns a vector composed of the distinct values of the vector v.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string ("1 2 1 3 2 1 2");   /* v = [1 2 1 3 2 1 2] */
...
vec u = vec_unique (v);                     /* u = [1 2 3]         */
