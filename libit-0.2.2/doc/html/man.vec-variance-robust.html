vec_variance_robust

Name

vec_variance_robust -- robust computation of the variance of a vector

Synopsis

#include <it/vec.h>

double vec_variance_robust( vec v );

DESCRIPTION

The function vec_variance_robust uses Knuth's two-pass algorithm for robust computation of the variance of vector v. During the first pass, a robust estimate of the mean is computed. During the second pass, we apply Knuth's method on the residuals. Use it when desesperate or in case of numerically strange values. The estimate given by this function is known to behave gracefully even with numerically strange data.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string( "5 -2 2 4 1" );  /* Create a vector of 5 elements */
double d = vec_variance_robust( v );     /* d = 6                         */
