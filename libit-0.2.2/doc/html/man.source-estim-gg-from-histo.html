source_estim_GG_from_histo

Name

source_estim_GG_from_histo -- estimate the parameters of a generalized Gaussian

Synopsis

#include <it/source_fit.h>

void source_estim_GG_from_histo( vec pdf, vec symbols, double * alpha, double * beta );

DESCRIPTION

This function estimates the parameters of a generalized Gaussian distribution (GGD) by minimizing the Kullback-Leibler distance between the estimated GGD and the histogram pdf. The values associated to each symbol of the alphabet are given in symbols. Therefore, symbols has the same size as pdf, which must sum to one. The estimated alpha and beta parameters of the GGD are returned at the address pointed by the alpha and beta arguments. These arguments are also used as the initial GGD parameters for the minimization.

The generalized Gaussian distribution is defined as GGD(x) = exp(-(|x| / alpha)**beta).
