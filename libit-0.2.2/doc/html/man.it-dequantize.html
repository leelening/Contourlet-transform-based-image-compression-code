it_dequantize

Name

it_dequantize -- dequantize a single index

Synopsis

#include <it/quantizer.h>

double it_dequantize( it_quantizer_t *q, int i );

DESCRIPTION

This function dequantizes the index i using the dequantizer q and returns the reconstructed value.
