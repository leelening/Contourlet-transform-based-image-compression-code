mat_new

Name

mat_new -- Create a new matrix of doubles

Synopsis

#include <it/mat.h>

mat mat_new( idx_t h, idx_t w );

DESCRIPTION

The mat_new function is used to create a new matrix of size hxw elements of double, where h and w respectively denote the number of rows and columns. New matrix elements have indetermined value.

EXAMPLE

#include <mat.h>

...

mat m = mat_new (2,2); 

mat_zeros (m); 

/*
m = 
[[ 0   0 ]
 [ 0   0 ]]
*/
