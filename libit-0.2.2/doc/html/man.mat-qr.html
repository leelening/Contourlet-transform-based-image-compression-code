mat_qr

Name

mat_qr -- performs in-place QR decomposition of input matrix

Synopsis

#include <it/linalg.h>

vec mat_qr( mat A );

DESCRIPTION

The function mat_qr performs in-place QR decomposition of the matrix A. A matrix can be decomposed into A=QR where Q is square and orthogonal and R upper trapezoidal. We output the diagonal of matrix R.
