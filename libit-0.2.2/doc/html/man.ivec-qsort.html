ivec_qsort

Name

ivec_qsort -- Sort a vector with the quick sort

Synopsis

#include <it/vec.h>

void ivec_qsort( ivec v );

DESCRIPTION

The function ivec_sort sorts a vector with the quick sort algorithm. Note that it makes use of the standard C function qsort.

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string ("1 0 2");   /* v = [1 0 2] */
ivec_qsort (v);                       /* v = [0 1 2] */
