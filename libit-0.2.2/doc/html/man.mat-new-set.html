mat_new_set

Name

mat_new_set -- Return a matrix initialized with a value

Synopsis

#include <it/mat.h>

mat mat_new_set( double val, int h, int w );

DESCRIPTION

The function mat_new_set returns a matrix of size hxw, where h and w are respectively the height and the width of the return matrix. All the elements are set to the parameter value val.
