vec_sqr

Name

vec_sqr -- put the components of a vector to square

Synopsis

#include <it/vec.h>

void vec_sqr( vec v );

DESCRIPTION

The function vec_sqr squares all the components of v.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string("[1 2 3 4]");   /* vector [1 2 3 4] */
vec_sqr( v );                          /* v = [1 4 9 16]    */
