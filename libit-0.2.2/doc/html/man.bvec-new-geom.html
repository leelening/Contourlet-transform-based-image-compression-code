bvec_new_geom

Name

bvec_new_geom -- create a geometric sequence

Synopsis

#include <it/vec.h>

bvec bvec_new_geom( byte start, byte r, size_t N );

DESCRIPTION

The bvec_new_geom function creates a vector generated so that the elements correspond to the geometric sequence of starting value start, generated by the factor of r and of length N.

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_geom(1,2,6); /* creates the vector [1 2 4 8 16 32] */ 
bvec_delete(v);                /* free the vector                    */
