mat_distance_norm

Name

mat_distance_norm -- norm distance between two matrices

Synopsis

#include <it/distance.h>

double mat_distance_norm( mat m1, mat m2, double norm );

DESCRIPTION

This function computes the distance between the two matrices m1 and m2 corresponding to the norm norm. This norm is defined as the sum of the absolute differences between elements of m1 and the corresponding elements of m2 raised to the power of norm.

EXAMPLE

#include <distance.h>

...

mat m1 = mat_new(2,2);
mat m2 = mat_new(2,2);
m1[0][0] = 3.5;
m1[0][1] = 6.8;
m1[1][0] = 2.2;
m1[1][1] = 5.1;
m2[0][0] = 5.5;
m2[0][1] = 2.3;
m2[1][0] = 7.7;
m2[1][1] = 0.6;
/* compute the Euclidean distance between the two matrices */
double distance = mat_distance_norm(m1, m2, 2);
