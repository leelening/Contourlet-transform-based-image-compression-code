source_gaussian

Name

source_gaussian -- gaussian source generator

Synopsis

#include <it/source.h>

vec source_gaussian( size_t size, double mean, double std );

DESCRIPTION

This function generates a random continuous vector of length size where each sample is drawn from a Gaussian distribution of mean mean and standard deviation std.

EXAMPLE

#include <source.h>

...

vec v = source_gaussian(10, 0.0, 1.0);
