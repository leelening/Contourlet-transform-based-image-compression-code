vec_new_add

Name

vec_new_add -- Add two vectors

Synopsis

#include <it/vec.h>

vec vec_new_add( vec v1, vec v2 );

DESCRIPTION

The function vec_new_add adds the vector v2 to the vector v1. The result is stored in a new vector, which is returned. Note that the vectors v1 and v2 must have the same length.

EXAMPLE

#include <vec.h>

...

v1 = vec_new_string( "1 0.5 -2 5" );   /* vector [1 0.5 -2 5] */
v2 = vec_new_string( "-3 0 1 5" );     /* vector [-3 0 1 5]   */
v  = vec_new_add( v1, v2 );            /* v = [-2 0.5 -1 10]  */
vec_delete(v1);
vec_delete(v2);
vec_delete(v);
