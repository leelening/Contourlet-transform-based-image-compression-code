imat_ivec_mul

Name

imat_ivec_mul -- return the product of a vector by a matrix

Synopsis

#include <it/mat.h>

ivec imat_ivec_mul( imat m, ivec v );

DESCRIPTION

The function imat_ivec_mul returns the product of the matrix m by the vector v

EXAMPLE

#include <mat.h>

...

imat m = imat_new_ones (3,3);
ivec v = ivec_new_ones (3);
ivec r = imat_ivec_mul (m, v);  /* r = [3 3 3] */
