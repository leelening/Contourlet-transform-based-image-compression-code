ivec_1N

Name

ivec_1N -- set the vector to [1 2 3 ...]

Synopsis

#include <it/vec.h>

void ivec_1N( ivec v );

DESCRIPTION

he function ivec_1N sets the vector v to [1 2 3... ] so that the vector length remains the same.

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new(5); /* creates the vector (not initialized) */
ivec_1N(v);           /* set v to [1 2 3 4 5]                 */
ivec_delete(v);       /* free the vector                      */
