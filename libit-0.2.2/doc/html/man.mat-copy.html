mat_copy

Name

mat_copy -- Copy matrix elements to another

Synopsis

#include <it/mat.h>

void mat_copy( mat dest, mat orig );

DESCRIPTION

The function mat_copy copies elements from orig to matrix dest. Matrix dest must have same or greater dimensions than orig. If dest is bigger than orig, elements are copied to dest starting from [0,0].

EXAMPLE

#include <mat.h>

...

mat m1 = mat_new_zeros (2,2); 
mat m2 = mat_new (3,3);

mat_set (m2,1.); 

mat_copy (m2,m1); 

/*
m2 = 
[[ 0   0   1 ] 
 [ 0   0   1 ]
 [ 1   1   1 ]]
*/
