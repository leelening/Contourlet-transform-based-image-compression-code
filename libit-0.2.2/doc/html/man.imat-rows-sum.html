imat_rows_sum

Name

imat_rows_sum -- Sum of rows

Synopsis

#include <it/mat.h>

ivec imat_rows_sum( imat m );

DESCRIPTION

The function imat_rows_sum returns the summations of every row of the matrix m. These summations are stored into a vector, which length correspond to the height of the matrix m.
