mat_svd

Name

mat_svd -- computes the SVD of the input matrix

Synopsis

#include <it/linalg.h>

vec mat_svd( mat M, mat U, mat V );

DESCRIPTION

The function mat_svd computes the singular value decomposition of the matrix M. If singular subspaces are not needed, simply set U and/or V to NULL. If U and/or V are needed, be sure to make proper memory allocation beforehand. This function is borrowed from the JAMA Java puyblic-domain package. See: http://math.nist.gov/javanumerics/jama/

EXAMPLE

#include <linalg.h>

...

mat m = mat_new( 3, 3 ); 
mat u = mat_new( 3, 3 ); 
mat v = mat_new( 3, 3 ); 

vec sv; 

m[0][0] = 1;   m[0][1] = 5;   m[0][2] = 2; 
m[1][0] = 2;   m[1][1] = 3;   m[1][2] = 5; 
m[2][0] = 3;   m[2][1] = 7;   m[2][2] = 3; 

sv = mat_svd (m1, u, v);
