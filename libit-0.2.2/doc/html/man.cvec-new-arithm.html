cvec_new_arithm

Name

cvec_new_arithm -- create an arithmetic sequence

Synopsis

#include <it/vec.h>

cvec cvec_new_arithm( cplx start, cplx incr, size_t N );

DESCRIPTION

The function cvec_new_arithm creates a vector generated so that the elements correspond to the arithmetic sequence of starting value start, increasing by a step of incr and of length N.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_arithm(0,cplx_I,4); /* creates the vector [0 i 2i 3i] */
cvec_delete(v);                       /* free the vector                */
