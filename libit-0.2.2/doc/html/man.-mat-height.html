Mat_height

Name

Mat_height -- Get the height of a matrix

Synopsis

#include <it/mat.h>

int Mat_height( Mat m );

DESCRIPTION

This macro returns the height of the matrix m.

EXAMPLE

#include <mat.h>

...

Mat m = Mat_new(float, 5, 7);
int h = Mat_height(m); /* h = 5 */
