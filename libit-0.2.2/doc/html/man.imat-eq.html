imat_eq

Name

imat_eq -- Test matrices for equality

Synopsis

#include <it/mat.h>

int imat_eq( mat m1, mat m2 );

DESCRIPTION

The function imat_eq tests if m1 and m2 are equal and return a non-zero integer if m1==m2, or zero otherwise.

EXAMPLE

#include <mat.h>

...

imat m1, m2;
m1 = imat_new_zeros(2,2);
m2 = imat_new_ones(2,2);
if(imat_eq(m1, m2))
  printf("the matrices are equal");
else
  printf("the matrices are not equal");
