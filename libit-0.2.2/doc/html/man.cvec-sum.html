cvec_sum

Name

cvec_sum -- Sum of a vector

Synopsis

#include <it/vec.h>

cplx cvec_sum( cvec v );

DESCRIPTION

The function cvec_sum returns the sum of the vector v

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string( "i 1-2i 2.5 0 -i" ); /* Create a vector of 5 elements */
cplx s = cvec_sum( v );                        /* s = 3.5-2i                    */
