mat_set_between

Name

mat_set_between -- Set some matrix components to a value

Synopsis

#include <it/mat.h>

mat mat_set_between( mat m, size_t r1, size_t c1, size_t r2, size_t c2, double val );

DESCRIPTION

The mat_set_between function sets all the elements of the matrix m whose position stands in the index interval [r1,r2]x[c1,c2] to the value val. See also: vec_set_between

EXAMPLE

#include <mat.h>

...

mat m = mat_new_zeros (3,3); 

mat_set_between (m, 1, 1, 2, 2, 1.); 

/*
m = 
[[0   0   0]
 [0   1   1]
 [0   1   1]]
*/
