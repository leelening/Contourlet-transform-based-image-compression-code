modulate_bpsk

Name

modulate_bpsk -- BPSK modulation

Synopsis

#include <it/channel.h>

vec modulate_bpsk( bvec v );

DESCRIPTION

This function modulates the binary vector v to a real vector using binary phase shift keying (BPSK) modulation. The 0 bit is represented with the real value -1, whereas the 1 bit is represented with the real value +1.

EXAMPLE

#include <channel.h>

...

bvec message = bvec_new_string("0 1 1 0 0 0 1");

/* modulate the message using BPSK */
vec modulated = modulate_bpsk(output);
