it_dwt2D

Name

it_dwt2D -- separable 2D discrete wavelet transform

Synopsis

#include <it/wavelet2D.h>

mat it_dwt2D( mat m, it_wavelet_lifting_t const *lifting, int levels );

DESCRIPTION

This function computes the levels-level separable discrete wavelet transform of m using the lifting steps described by lifting. Currently the lifting steps for the usual 5/3 and 9/7 biorthogonal filters are predefined as "it_wavelet_lifting_53" and "it_wavelet_lifting_97", although you can provide your own lifting steps. This transform is critically sampled, therefore the output matrix of coefficients has the same size as the input matrix m.

EXAMPLE

#include <wavelet2D.h>

...

/* analyse a signal 'm' using a 5-level 9/7 wavelet decomposition */
mat mt = it_dwt2D(m, it_wavelet_lifting_97, 5);
/* recompose the signal by inverse transform */
mat mr = it_idwt2D(mt, it_wavelet_lifting_97, 5);
