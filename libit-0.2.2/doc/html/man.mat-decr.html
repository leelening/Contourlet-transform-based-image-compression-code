mat_decr

Name

mat_decr -- Decrease the elements of a matrix by a constant

Synopsis

#include <it/mat.h>

void mat_decr( mat m, double a );

DESCRIPTION

The function mat_decr substracts the value a from every element of m.

EXAMPLE

#include <mat.h>

...

mat m = mat_new_zeros (2,2); 

mat_decr (m,2.);

/*
m = 
[[ -2   -2 ]
 [ -2   -2 ]]
*/
