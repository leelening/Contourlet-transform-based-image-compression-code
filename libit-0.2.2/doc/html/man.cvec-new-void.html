cvec_new_void

Name

cvec_new_void -- create an empty vector

Synopsis

#include <it/vec.h>

cvec cvec_new_void( );

DESCRIPTION

The function cvec_new_void creates a new empty vector, i.e. a vector of length 0.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_void();    /* creates the vector [] */
cvec_delete(v);              /* free the vector       */
