it_quantize_mat

Name

it_quantize_mat -- quantize a matrix

Synopsis

#include <it/quantizer.h>

imat it_quantize_mat( it_quantizer_t *q, mat m );

DESCRIPTION

This function quantizes the matrix m by processing it row by row. Each row is vector quantized using the quantizer q. The matrix with rows corresponding to the quantized index vectors is returned.
