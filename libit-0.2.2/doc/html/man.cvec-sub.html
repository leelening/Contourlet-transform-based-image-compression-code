cvec_sub

Name

cvec_sub -- substract a vector from another

Synopsis

#include <it/vec.h>

void cvec_sub( cvec v1, cvec v2 );

DESCRIPTION

The function cvec_sub substracts the vector v2 to the vector v1. The result is stored in the vector v1. Note that the vectors v1 and v2 must have the same length.

EXAMPLE

#include <vec.h>

...

v1 = ivec_new_string( "i i -2 5" );    /* vector [i i -2 5]    */
v2 = ivec_new_string( "-3 1 1 5" );    /* vector [-3 1 1 5]    */
ivec_sub( v1, v2 );                    /* v1 = [i+3 i-1 -3 0]  */
ivec_delete(v1);
ivec_delete(v2);
