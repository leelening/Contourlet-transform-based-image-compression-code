ivec_new_string

Name

ivec_new_string -- Create a vector from a string

Synopsis

#include <it/io.h>

ivec ivec_new_string( char * s );

DESCRIPTION

The function ivec_new_string creates a vector from the string s. The string is parsed and uses the following characters as element separators: space and comma. The function allocate some memory.

EXAMPLE

#include <io.h>

...

ivec v1 = ivec_new_string("0 1 2 3");     /* Vector [0 1 2 3] */
ivec v2 = ivec_new_string("[2,1,2,1]");   /* Vector [2 1 2 1] */
