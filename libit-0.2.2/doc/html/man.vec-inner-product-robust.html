vec_inner_product_robust

Name

vec_inner_product_robust -- robust computation of inner product

Synopsis

#include <it/vec.h>

double vec_inner_product_robust( vec v1, vec v2 );

DESCRIPTION

The function vec_inner_product_robust uses Kahan's method to compute robust inner product between the two input vectors v1 and v2. These vectors must have the same length.

EXAMPLE

#include <vec.h>

...

vec v1 = vec_new_string("-1 -2 1.5");        /* vector [ -1 -2 1.5 ]  */
vec v2 = vec_new_string("2 0 4");            /* vector [ 2 0 4 ]      */
double p = vec_inner_product_robust(v1, v2); /* p = 4                 */
