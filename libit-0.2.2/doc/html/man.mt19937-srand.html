mt19937_srand

Name

mt19937_srand -- seeds the MT19937 PRNG

Synopsis

#include <it/random.h>

void mt19937_srand( unsigned int seed );

DESCRIPTION

This function seeds the MT19937 PRNG with seed seed. This is a 32 bit initialization. Like all other function having mt19937 in their name, it is intended to use advanced features of this generator. See mt19937_srand_by_array.

EXAMPLE

#include <random.h>

...

mt19937_srand ( 12345 );      /* init with seed 12345             */ 
mt19937_rand_int32 ();        /* get first number from seed above */
