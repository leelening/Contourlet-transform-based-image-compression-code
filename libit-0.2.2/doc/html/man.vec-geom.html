vec_geom

Name

vec_geom -- set the vector to a geometric sequence

Synopsis

#include <it/vec.h>

void vec_geom( vec v, double start, double r );

DESCRIPTION

The function vec_geom sets the vector v so that the elements correspond to the geometric sequence of starting value start, generated by the geometric factor r. The length is kept unchanged.

EXAMPLE

#include <vec.h>

...

vec v = vec_new(5);  /* creates the vector (not initialized) */
vec_geom(v,1.5,2);   /* set v to [1.5 3 6 12 24]             */
vec_delete(v);       /* free the vector                      */
