Mat_width

Name

Mat_width -- get the width of a matrix

Synopsis

#include <it/mat.h>

int Mat_width( Mat m );

DESCRIPTION

This macro returns the width of the matrix m.

EXAMPLE

#include <mat.h>

...

Mat m = Mat_new(float, 5, 7);
int w = Mat_width(m); /* w = 7 */
