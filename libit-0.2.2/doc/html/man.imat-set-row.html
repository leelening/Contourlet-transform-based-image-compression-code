imat_set_row

Name

imat_set_row -- set a matrix row to a constant value

Synopsis

#include <it/mat.h>

imat imat_set_row( imat m, int row, int value );

DESCRIPTION

This function sets an entire row row of the matrix m to the value value.

EXAMPLE

#include <mat.h>

...

imat m = imat_new_zeros(2,2);
imat_set_row(m, 1, 4); /* [ [ 0 4 ]
                            [ 0 4 ] ] */
