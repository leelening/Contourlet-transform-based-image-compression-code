vec_apply_function

Name

vec_apply_function -- Apply a function to all the elements of a vector

Synopsis

#include <it/vec.h>

void vec_apply_function( vec v, it_function_t function, it_args_t args );

DESCRIPTION

The function vec_apply_function applies a function to all the elements of the vector v. The second argument must be a function of type it_function_t, however a C function can be casted into a it_function_t using the macro IT_FUNCTION.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string ("1 4 9");

vec_apply_function (v, IT_FUNCTION (sqrt));  /* Cast the standard function sqrt */
                                             /* into a it_function_t type and   */
                                             /* call vec_apply_function         */

/* v = [1 2 3] */
