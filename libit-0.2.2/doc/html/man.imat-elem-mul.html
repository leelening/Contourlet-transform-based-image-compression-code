imat_elem_mul

Name

imat_elem_mul -- Performs element-wise matrix multiplication

Synopsis

#include <it/mat.h>

void imat_elem_mul( imat m1, imat m2 );

DESCRIPTION

The function imat_elem_mul multiplies element (i,j) of m1 by element (i,j) of m2 and stores the result in m1.
