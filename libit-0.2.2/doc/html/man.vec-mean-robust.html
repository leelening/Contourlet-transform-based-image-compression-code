vec_mean_robust

Name

vec_mean_robust -- robust computation of the mean of a vector

Synopsis

#include <it/vec.h>

double vec_mean_robust( vec v );

DESCRIPTION

The function vec_mean_robust uses Kahan's method to compute the artihmetic mean of a vector. Use it when desesperate. See vec_sum_robust, vec_mean

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string( "5 -2 2 4 1" );  /* Create a vector of 5 elements */
double d = vec_mean_robust( v );         /* d = 2                         */
