bvec_find_sorted

Name

bvec_find_sorted -- find the position of a value in a sorted vector

Synopsis

#include <it/vec.h>

idx_t bvec_find_sorted( bvec v, byte val );

DESCRIPTION

Assuming that the vector is sorted, the function bvec_find_sorted returns a position in the vector v where the value val can be found. If the value val is not in the vector, the function returns the position where is should be inserted to maintain the vector sorted. In particular, is the value val is greater than any element of the vector, the function returns the length of the vector.

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_arithm (1,2,4);      /* v =[1,3,5,7]   */

idx_t pos = bvec_find_sorted (v, 5);   /* pos = 2 */          
pos = bvec_find_sorted (v, 4);         /* pos = 2 */    
pos = bvec_find_sorted (v, 0);         /* pos = 0 */    
pos = bvec_find_sorted (v, 15);        /* pos = 4 */
