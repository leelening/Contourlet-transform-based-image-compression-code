cvec_set_length_max

Name

cvec_set_length_max -- Modify the amount of memory allocated to a vector

Synopsis

#include <it/vec.h>

void cvec_set_length_max( cvec v, size_t N );

DESCRIPTION

The cvec_set_length_max macro modifies the amount of memory allocated for a vector so that the maximum number of allocated elements is equal to N. It is the equivalent of cvec_new_alloc for existing vectors. This function may modify the pointer representing the vector.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new(10);    /* allocate a new vector of size 10      */
cvec_set_length_max(v,4); /* modify the amount of allocated memory */
cvec_delete(v);           /* free the vector                       */
