ivec_set_length_max

Name

ivec_set_length_max -- Modify the amount of memory allocated to a vector

Synopsis

#include <it/vec.h>

void ivec_set_length_max( ivec v, size_t N );

DESCRIPTION

The ivec_set_length_max macro modifies the amount of memory allocated for a vector. It is the equivalent of ivec_new_alloc for existing vectors. This function may modify the pointer representing the vector.

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new(10);    /* allocate a new vector of size 10      */
ivec_set_length_max(v,4); /* modify the amount of allocated memory */
ivec_delete(v);           /* free the vector                       */
