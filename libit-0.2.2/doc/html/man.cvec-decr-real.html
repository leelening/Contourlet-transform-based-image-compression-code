cvec_decr_real

Name

cvec_decr_real -- Substract a real value to the vector components

Synopsis

#include <it/vec.h>

void cvec_decr_real( cvec v, double a );

DESCRIPTION

The function vec_decr substracts the real a from all the components of the vector v. Hence the imaginary part is not modified by this function.

EXAMPLE

#include <vec.h>

...

cvec v  = cvec_new_string("0.3-i i 1");  /* vector [ 0.3-i 1+i i ]      */
cvec_decr_real(v, 1.3);                  /* v = [ -1-i -0.3+i -1.3+i ]  */
cvec_delete(v);
