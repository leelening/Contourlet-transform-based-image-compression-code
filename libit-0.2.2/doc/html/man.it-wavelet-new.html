it_wavelet_new

Name

it_wavelet_new -- wavelet transform constructor

Synopsis

#include <it/wavelet.h>

void it_wavelet_new( it_wavelet_lifting_t const *lifting, int level );

DESCRIPTION

The it_wavelet_new function creates a new wavelet transform object on level levels using lifting steps lifting. This object can later be used as an it_transform_t to perform discrete wavelet transform or inverse wavelet transform on a real-valued vector using the it_transform or it_itransform method respectively.

Creating this transform object and using the it_transform method is prefered to using it_dwt when multiple transform on vectors of the same size are to be done.

EXAMPLE

#include <wavelet.h>

...

it_wavelet_t *wavelet = it_wavelet_new(it_wavelet_lifting_97, 5);
vec v = vec_new_string("[ 0.3 1.5 2.5 ]");
vec vt;
vt = (vec) it_transform(wavelet, (vec) v);
