vec_variance

Name

vec_variance -- computes variance estimate of input vector

Synopsis

#include <it/vec.h>

double vec_variance( vec v );

DESCRIPTION

The function vec_variance computes an estimate of input vector v variance. See vec_variance_robust.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string( "5 -2 2 4 1" );  /* Create a vector of 5 elements */
double d = vec_variance( v );            /* d = 6                         */
