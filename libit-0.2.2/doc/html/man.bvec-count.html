bvec_count

Name

bvec_count -- Count an element in a vector

Synopsis

#include <it/vec.h>

int bvec_count( bvec v, byte a );

DESCRIPTION

The function bvec_count counts the number of times that the byte a occurs in the vector v and returns this value.

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_string( "1 3 -1 3" );    /* Vector [1 3 -1 3] */
int  c = bvec_count( v, 3 );               /* c = 2             */
