it_idft

Name

it_idft -- inverse discrete Fourier transform

Synopsis

#include <it/fourier.h>

cvec it_idft( cvec v );

DESCRIPTION

This function computes the inverse discrete Fourier transform of the complex vector v. The first element of this vector corresponds to the lowest frequency (DC value). If v is conjugate symmetric, that is if v[N-i] = v[i]*, where N is the length of v, the output will be a real vector (that is, all elements of the complex output vector will have a null imaginary part).

EXAMPLE

#include <fourier.h>

...

cvec vt = cvec_new_string("1 1+i 0 1-i");
/* reconstruct the vector using the inverse discrete Fourier transform */
cvec v = it_idft(vt);
