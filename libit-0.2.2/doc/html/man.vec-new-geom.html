vec_new_geom

Name

vec_new_geom -- create a geometric sequence of double values

Synopsis

#include <it/vec.h>

vec vec_new_geom( double start, double r, size_t N );

DESCRIPTION

The function vec_new_geom creates a vector generated so that the elements correspond to the geometric sequence of starting value start, generated by the factor of r and of length N.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_geom(1.5,2,5);/* creates the vector [1.5 3 6 12 24] */
vec_delete(v);                /* free the vector                    */
