imat_height_max

Name

imat_height_max -- Get current maximum height of input matrix

Synopsis

#include <it/mat.h>

int imat_height_max( imat m );

DESCRIPTION

The function imat_height_max returns the maximum height of a matrix. By maximum height, it means the height to which the matrix can be set without calling a free/malloc function.
