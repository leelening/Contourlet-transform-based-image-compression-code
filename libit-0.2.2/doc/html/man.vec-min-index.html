vec_min_index

Name

vec_min_index -- Position of the minimum in a vector

Synopsis

#include <it/vec.h>

int vec_min_index( vec v );

DESCRIPTION

The function vec_min_index returns the position where the minimum of the vector v is reached. If the minimum has a multiplicity which is greater than 1, then the first position should be returned.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string ("1 -2.7 7 0");         /* vector [1 -2.7 7 0] */
int p = vec_min_index (v);                     /* p = 1               */
