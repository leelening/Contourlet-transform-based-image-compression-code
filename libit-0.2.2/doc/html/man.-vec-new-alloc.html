Vec_new_alloc

Name

Vec_new_alloc -- create a new generic vector

Synopsis

#include <it/vec.h>

Vec Vec_new_alloc( size_t elem_size, size_t length, size_t length_max );

DESCRIPTION

The macro Vec_new_alloc creates a new vector of length entities. Unlike the macro Vec_new, the macro Vec_new_alloc allocates the memory for a greater number of elements than the vector length. Hence, the number length_max gives the size up to which the vector can grow without any memory reallocation. This function allows to avoid to reallocate memory if the vector comes to grow up to the value length_max. This may be useful if the vector is used as a stack. Note however that in most cases the user does not have to care about the real amount of memory allocated for the vector, since the geometric reallocation strategy allows to avoid to introduce additional complexity while modifying the size of a vector.
