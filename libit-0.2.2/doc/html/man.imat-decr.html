imat_decr

Name

imat_decr -- Decrease the elements of a matrix by a constant

Synopsis

#include <it/mat.h>

void imat_decr( imat m, int a );

DESCRIPTION

The function imat_decr substracts the value a from every element of m

EXAMPLE

#include <mat.h>

...

imat m = imat_new_zeros (2,2);

imat_decr (m,-2);

/*
m = 
[[ 2   2 ]
 [ 2   2 ]]
*/
