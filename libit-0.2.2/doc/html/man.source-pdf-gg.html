source_pdf_GG

Name

source_pdf_GG -- discrete GGD generator

Synopsis

#include <it/source_fit.h>

vec source_pdf_GG( vec symbols, double alpha, double beta );

DESCRIPTION

This function generates an approximate of the generalized Gaussian distribution for a discrete source taking values in the symbols alphabet. The alpha and beta arguments are the parameters of the generalized Gaussian distribution, which is defined as GGD(x) = exp(-(|x| / alpha)**beta).
