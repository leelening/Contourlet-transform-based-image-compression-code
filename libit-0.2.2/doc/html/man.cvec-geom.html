cvec_geom

Name

cvec_geom -- set the vector to a geometric sequence

Synopsis

#include <it/vec.h>

void cvec_geom( cvec, cplx start, cplx r );

DESCRIPTION

The function cvec_geom sets the vector v so that the elements correspond to the geometric sequence of starting value start, generated by the complex geometric factor r. The length is kept unchanged.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new(5);  /* creates the vector (not initialized) */
cvec_geom(v,1,i);      /* set v to [1 i -1 -i 1]             */
cvec_delete(v);        /* free the vector                      */
