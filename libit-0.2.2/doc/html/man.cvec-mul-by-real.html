cvec_mul_by_real

Name

cvec_mul_by_real -- multiply a complex vector by a real constant

Synopsis

#include <it/vec.h>

void cvec_mul_by_real( cvec v, double a );

DESCRIPTION

The function cvec_mul_by multiplies each component of the vector v by the double floating point value a.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string ("1 i 3");   /* v = [1 i 3]   */
cvec_mul_by (v, 3);                   /* v = [3 3i 9]  */
