cmat_new_inv

Name

cmat_new_inv -- computes the inverse of the input matrix

Synopsis

#include <it/linalg.h>

cmat cmat_new_inv( cmat m );

DESCRIPTION

The function cmat_new_inv is the complex-analogon for mat_new_inv. It has currently the same behaviour as cmat_inv except that it outputs a newly created matrix when possible.
