ivec_range

Name

ivec_range -- set the vector to [0 1 2 3 ...]

Synopsis

#include <it/vec.h>

ivec ivec_range( ivec v );

DESCRIPTION

The function ivec_range sets the vector v to [0 1 2... ] so that the vector length remains the same.

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new(5); /* creates the vector (not initialized) */
ivec_range(v);        /* set v to [0 1 2 3 4]                 */
ivec_delete(v);       /* free the vector                      */
