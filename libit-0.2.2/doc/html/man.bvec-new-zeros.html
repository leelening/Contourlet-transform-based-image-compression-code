bvec_new_zeros

Name

bvec_new_zeros -- create a byte vector of zeros

Synopsis

#include <it/vec.h>

bvec bvec_new_zeros( size_t N );

DESCRIPTION

The function bvec_new_zeros creates a new vector of N bytes initialized to the value 0.

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_zeros(5);  /* creates the vector [0 0 0 0 0] */
bvec_delete(v);              /* free the vector                */
