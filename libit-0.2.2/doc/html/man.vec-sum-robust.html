vec_sum_robust

Name

vec_sum_robust -- Robut summation over elements of a vector

Synopsis

#include <it/vec.h>

double vec_sum_robust( vec v );

DESCRIPTION

The function vec_sum_robust uses Kahan's method to provide numerically robust summation over elements of a vector. It is known to minimize roundoff error in fixed-arithmetic. Use it when desesperate.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string( "5 -2 2 4 1" );  /* Create a vector of 5 elements */
double d = vec_sum_robust( v );          /* d = 10                        */
