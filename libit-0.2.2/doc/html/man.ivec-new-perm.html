ivec_new_perm

Name

ivec_new_perm -- generates an uniformly shuffled ivec

Synopsis

#include <it/vec.h>

ivec ivec_new_perm( size_t len, unsigned int seed );

DESCRIPTION

The function ivec_new_perm generates uniformly distributed permutations (or shuffles) using the Fisher-Yates algorithm. The newly created permutation has size len and is generated initializing the PRNG with seed. It is basically the opposite of function ivec_qsort. Reference: R.A. Fisher and F. Yates, Example 12, Statistical Tables, London, 1938.

EXAMPLE

#include <vec.h>

...

ivec p = ivec_new_perm (5,1234);   /* p= [0 2 3 4 1] */ 
ivec_delete (p);
p = ivec_new_perm (5, 12345);      /* p= [4 0 2 3 1] */
