mat_diag

Name

mat_diag -- Set a matrix to a diagonal matrix

Synopsis

#include <it/mat.h>

void mat_diag( mat m, vec v );

DESCRIPTION

The function mat_diag sets a matrix m to a diagonal matrix, i.e. a matrix whose elements are equals to 0 excepted the diagonal which is defined by the vector parameter v

EXAMPLE

#include <mat.h>

...

mat m = mat_new_ones (2,2);   /* m = [[1 1]  */
                              /*      [1 1]  */

vec d = vec_new_set (3,2);    /* d = [2 2]   */

mat_diag (m, v);              /* m = [[3 0]  */
                              /*      [0 3]  */
