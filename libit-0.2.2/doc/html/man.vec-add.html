vec_add

Name

vec_add -- add a vector to another

Synopsis

#include <it/vec.h>

void vec_add( vec v1, vec v2 );

DESCRIPTION

The function vec_add adds the vector v2 to the vector v1. The result is stored in the vector v1. Note that the vectors v1 and v2 must have the same length.

EXAMPLE

#include <vec.h>

...

v1 = vec_new_string( "1 0.5 -2 5" );   /* vector [1 0.5 -2 5] */
v2 = vec_new_string( "-3 0 1 5" );     /* vector [-3 0 1 5]   */
vec_add( v1, v2 );                     /* v1 = [-2 0.5 -1 10] */
vec_delete(v1);
vec_delete(v2);
