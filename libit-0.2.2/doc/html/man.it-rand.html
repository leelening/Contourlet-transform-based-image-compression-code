it_rand

Name

it_rand -- uniform random number generator

Synopsis

#include <it/random.h>

double it_rand( void );

DESCRIPTION

This function generates a single random value drawn in the range [0,1] from a uniform distribution.

EXAMPLE

#include <random.h>
