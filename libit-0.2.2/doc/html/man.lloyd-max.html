lloyd_max

Name

lloyd_max -- generate the Lloyd-Max quantizer codebook

Synopsis

#include <it/quantizer.h>

void lloyd_max( it_function_t function, it_args_t args, double a, double b, int N); );

DESCRIPTION

Generate the codebook for the Lloyd-Max quantizer on N levels. This is the optimal codebook for a stationnary i.i.d. source. [a,b] is an initial interval where to expect the pdf function to be non-zero. A good guess is [ mean - std_dev, mean + std_dev ]. Note that returned centroids may lie outside this interval. The algorithm stops when the relative difference in distortion is smaller than the IT_EPSILON constant. The parameter args is used to specify the extra parameters of the pdf function function.

EXAMPLE

#include <quantizer.h>

...

it_function_args(itf_gaussian) itf_gaussian_args;
vec codebook;
itf_gaussian_args.sigma = 1.0;
codebook = lloyd_max(itf_gaussian, &itf_gaussian_args, -5, 5, 4);
