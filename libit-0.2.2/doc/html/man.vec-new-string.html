vec_new_string

Name

vec_new_string -- Create a vector from a string

Synopsis

#include <it/io.h>

vec vec_new_string( char * s );

DESCRIPTION

The function vec_new_string creates a vector from the string s. The string is parsed and uses the following characters as element separators: space and comma. The function allocate some memory.

EXAMPLE

#include <io.h>

...

vec v1 = vec_new_string("0 1 2 3");     /* Vector [0 1 2 3] */
vec v2 = vec_new_string("[2,1,2,1]");   /* Vector [2 1 2 1] */
