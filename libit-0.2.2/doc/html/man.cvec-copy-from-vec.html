cvec_copy_from_vec

Name

cvec_copy_from_vec -- copy a vector with a type conversion

Synopsis

#include <it/vec.h>

void cvec_copy_from_vec( cvec dest, vec orig );

DESCRIPTION

The function copies the orig vector into the dest vector by making a type conversion. Note that the vectors must be already allocated and must have the same size.
