cmul

Name

cmul -- Multiply two complex numbers

Synopsis

#include <it/cplx.h>

cplx cmul( cplx a, cplx b );

DESCRIPTION

The function cmul computes and returns the product of the complex numbers a and b.

EXAMPLE

#include <cplx.h>

...

cplx a = cplx(1,1);
cplx b = cplx(2,3);
cplx c = cmul(a, b);
