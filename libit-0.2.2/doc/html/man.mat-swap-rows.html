mat_swap_rows

Name

mat_swap_rows -- swaps two rows in input matrix

Synopsis

#include <it/mat.h>

void mat_swap_rows( mat m, idx_t i, idx_t j );

DESCRIPTION

The function mat_swap_rows swaps rows i and j in matrix m.

EXAMPLE

#include <mat.h>

...

mat m = mat_new (2,2); 

m[0][0] = 1; m[0][1] = 3;
m[1][0] = 2; m[1][1] = 4; 

mat_swap_rows (m, 0, 1); 

/*
m = 
[[ 2   4
   1   3 ]]
*/
