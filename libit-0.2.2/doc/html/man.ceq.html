ceq

Name

ceq -- Check if two complex numbers are equal

Synopsis

#include <it/cplx.h>

int ceq( cplx a, cplx b );

DESCRIPTION

The function ceq check if the two complex numbers are identical.

EXAMPLE

#include <cplx.h>

...

cplx a = cplx(1,1);
cplx b = cplx(2,3);
if(ceq(a, b))
  printf("a and b are equal
");
