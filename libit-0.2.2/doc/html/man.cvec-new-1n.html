cvec_new_1N

Name

cvec_new_1N -- create the vector 1, 2, ...

Synopsis

#include <it/vec.h>

cvec cvec_new_1N( size_t N );

DESCRIPTION

The function cvec_new_1N creates the vector [1 2 ... N], where the value N denotes the length of the vector.

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_1N(5);    /* creates the vector [1 2 3 4 5] */
cvec_delete(v);             /* free the vector                */
