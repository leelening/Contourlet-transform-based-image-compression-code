cvec_clone

Name

cvec_clone -- create a copy of a complex vector

Synopsis

#include <it/vec.h>

cvec cvec_clone( cvec v );

DESCRIPTION

This function creates a copy of the vector v. Note that it is not valid to use the operator = instead.

EXAMPLE

#include <vec.h>

...

cvec a = cvec_new_string ("i 1 2 3");   /* a = [i 1 2 3] */
cvec b = cvec_clone (b);                /* b = [i 1 2 3] */
