cmat_copy

Name

cmat_copy -- Copy a matrix from another

Synopsis

#include <it/mat.h>

void cmat_copy( cmat dest, cmat orig );

DESCRIPTION

The function cmat_copy copies elements from orig to matrix dest. Matrix dest must have same or greater dimensions than orig. If dest is bigger than orig, elements are copied to dest starting from [0,0].

#include <mat.h>
