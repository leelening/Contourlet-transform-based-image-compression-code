vec_qsort

Name

vec_qsort -- Sort a vector with the quick sort

Synopsis

#include <it/vec.h>

void vec_qsort( vec v );

DESCRIPTION

The function vec_sort sorts a vector with the quick sort algorithm. Note that it makes use of the standard C function qsort.

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string ("1 0 2");    /* v = [1 0 2] */
vec_qsort (v);                       /* v = [0 1 2] */
