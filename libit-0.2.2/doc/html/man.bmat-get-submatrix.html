bmat_get_submatrix

Name

bmat_get_submatrix -- Extract submatrix from input matrix

Synopsis

#include <it/mat.h>

bmat bmat_get_submatrix( bmat m, idx_t r1, idx_t c1, idx_t r2, idx_t c2 );

DESCRIPTION

The function bmat_get_submatrix creates a new matrix containing elements of m in range [r1,c1]x[r2,c2].
