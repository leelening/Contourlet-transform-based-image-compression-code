ivec_clone

Name

ivec_clone -- create a copy of a vector

Synopsis

#include <it/vec.h>

ivec ivec_clone( ivec v );

DESCRIPTION

This function creates a copy of the vector v. Note that it is not valid to use the operator = instead.

EXAMPLE

#include <vec.h>

...

ivec a = ivec_new_string ("0 1 2 3");   /* a = [0 1 2 3] */
ivec b = ivec_clone (b);                /* b = [0 1 2 3] */
