mat_cond

Name

mat_cond -- computes the condition number of the input matrix

Synopsis

#include <it/linalg.h>

double mat_cond( mat A );

DESCRIPTION

The function mat_cond returns the condition number of the matrix A. Since it is the ratio between the two extremal singular values, we therefore proceed to a SVD beforehand.

EXAMPLE

#include <linalg.h>

...

mat m1 = mat_new( 3, 3 ); 
double c; 

m1[0][0] = 1;   m1[0][1] = 5;   m1[0][2] = 2; 
m1[1][0] = 2;   m1[1][1] = 3;   m1[1][2] = 5; 
m1[2][0] = 3;   m1[2][1] = 7;   m1[2][2] = 3; 

c = mat_cond (m1);         /* c = 13.185340 */
