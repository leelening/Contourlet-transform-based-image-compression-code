bmat_void

Name

bmat_void -- Set a matrix to void

Synopsis

#include <it/mat.h>

bmat bmat_void( bmat m );

DESCRIPTION

The function bmat_void sets a matrix to the void matrix, i.e. a matrix containing no element.

EXAMPLE

#include <mat.h>

...

bmat m = bmat_new_ones(2,2);  /* Matrix of size 2x2                           */
bmat_void( m );               /* Matrix does not contains any element anymore */
