imat_vec_mul

Name

imat_vec_mul -- return the product of a vector by a matrix

Synopsis

#include <it/mat.h>

vec imat_vec_mul( imat m, vec v );

DESCRIPTION

The function imat_vec_mul returns the product of the matrix m by the vector v

EXAMPLE

#include <mat.h>

...

imat m = imat_new_ones (3,3);
vec v = vec_new_ones (3);
vec r = imat_vec_mul (m, v);  /* r = [3 3 3] */
