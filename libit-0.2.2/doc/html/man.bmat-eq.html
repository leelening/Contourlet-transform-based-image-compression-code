bmat_eq

Name

bmat_eq -- Test matrices for equality

Synopsis

#include <it/mat.h>

int bmat_eq( bmat m1, bmat m2 );

DESCRIPTION

The function bmat_eq tests if m1 and m2 are equal and return a non-zero integer if m1==m2, or zero otherwise.

EXAMPLE

#include <mat.h>

...

bmat m1, m2;
m1 = bmat_new_zeros(2,2);
m2 = bmat_new_ones(2,2);
if(bmat_eq(m1, m2))
  printf("the matrices are equal");
else
  printf("the matrices are not equal");
