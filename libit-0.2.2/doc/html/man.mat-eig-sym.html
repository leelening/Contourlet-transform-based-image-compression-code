mat_eig_sym

Name

mat_eig_sym -- computes eigen decomposition of a symmetrix matrix

Synopsis

#include <it/linalg.h>

vec mat_eig_sym( mat a, mat evec );

DESCRIPTION

The function mat_eig_sym performs eigen-decomposition of a real symmetric matrix a. Eigenvectors are stored in input matrix evec. Make sure to make proper memory allocation beforehand. Eigenvalues are output of the function. This function is borrowed from the JAMA Java public-domain package. See: http://math.nist.gov/javanumerics/jama/

EXAMPLE

#include <linalg.h>

...

m1 = mat_new( 3, 3 ); 
mat evec = mat_new (3, 3);
vec eval; 

m1[0][0] = 1;   m1[0][1] = 5;   m1[0][2] = 2; 
m1[1][0] = 5;   m1[1][1] = 3;   m1[1][2] = 5; 
m1[2][0] = 2;   m1[2][1] = 5;   m1[2][2] = 3; 

eval = mat_eig_sym (a, evec);

/*
eval = [10.639189 -3.715090 0.075900]
evec = 
[[0.469034 0.607644 0.640918]
 [0.677681 -0.712979 0.180026]
 [0.566353 0.349900 -0.746200]]
*/
