mt19937_rand_res53

Name

mt19937_rand_res53 -- generates an uniform pseudo-random real number

Synopsis

#include <it/random.h>

double mt19937_rand_res53( void );

DESCRIPTION

This function generates an uniformly-distributed real number in the [0,1)-real-interval with 53-bit resolution. Like all other functions having mt19937 in their name, they are intended to use advanced features of the MT19937.

EXAMPLE

#include <random.h>

...

mt19937_srand( 12345 ); 
double r = mt19937_rand_res53(); /* first real number from above seed */
