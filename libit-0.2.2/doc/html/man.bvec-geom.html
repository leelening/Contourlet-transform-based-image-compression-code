bvec_geom

Name

bvec_geom -- set the vector to a geometric sequence

Synopsis

#include <it/vec.h>

void bvec_geom( bvec v, byte start, byte r );

DESCRIPTION

The function bvec_geom sets the vector v so that the elements correspond to the arithmetic sequence of starting value start, generated by the geometric factor r. The length is kept unchanged.

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new(5); /* creates the vector (not initialized) */
bvec_geom(v,1,3);     /* set v to [1 3 9 27 81]               */
bvec_delete(v);       /* free the vector                      */
