mat_is_symmetric

Name

mat_is_symmetric -- tests whether input matrix is symmetric or not

Synopsis

#include <it/linalg.h>

int mat_is_symmetric( mat A );

DESCRIPTION

The function mat_is_symmetric returns 1 if A is symmetric, otherwise it returns 0.

EXAMPLE

#include <linalg.h>

...

mat a= mat_new (2,2); 
int s; 

a[0][0] = 12.;   a[0][1] = 2.; 
a[1][0] = 2.;   a[1][1] = 11.; 

s = mat_is_symmetric( a ); /* s = 1 */
