it_uniform_quantizer_new_from_center

Name

it_uniform_quantizer_new_from_center -- create a uniform scalar quantizer from center

Synopsis

#include <it/quantizer.h>

void it_uniform_quantizer_new_from_center( double center, double step, double factor );

DESCRIPTION

This function creates a new uniform scalar quantizer object from the center of the quantization codebook center (i.e. the reconstruction value corresponding to the symbol 0) and the stepsize step. The factor arguments is used to create a dead-zone around the central reconstruction value, that is the bin for the 0 symbol is factor times the size of the bin for other symbols. For classical uniform quantization it is set to 1.0, whereas dead-zone quantization usually refers to factor=2.0.

RETURN VALUE

EXAMPLE

#include <quantizer.h>

...

/* create a new uniform quantizer with step 1/256 centered on 0
   with a dead-zone of twice the step size                      */
it_uniform_quantizer_t uniq =
      it_uniform_quantizer_new_from_center(0, 1/256., 2.0);
