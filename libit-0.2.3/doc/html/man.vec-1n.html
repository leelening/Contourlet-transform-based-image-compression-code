vec_1N

Name

vec_1N -- set the vector to [1 2 3 ...]

Synopsis

#include <it/vec.h>

void vec_1N( vec v );

DESCRIPTION

The function vec_1N sets the vector v to [1 2 3... ] so that the vector length remains the same.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new(5); /* creates the vector (not initialized) */
vec_1N(v);          /* set v to [1 2 3 4 5]                 */
vec_delete(v);      /* free the vector                      */
