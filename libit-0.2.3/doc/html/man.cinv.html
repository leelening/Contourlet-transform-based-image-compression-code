cinv

Name

cinv -- Inverse a complex number

Synopsis

#include <it/cplx.h>

cplx cinv( cplx a );

DESCRIPTION

The function cinv returns the inverse of a, if it exists.

RETURN VALUE

The inverse of the complex

EXAMPLE

#include <cplx.h>

...

cplx a = cplx(1,1);
cplx c = cinv(a);
