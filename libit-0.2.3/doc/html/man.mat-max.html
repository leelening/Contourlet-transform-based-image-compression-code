mat_max

Name

mat_max -- Return the maximum value of input matrix

Synopsis

#include <it/mat.h>

double mat_max( mat m );

DESCRIPTION

The function mat_max returns the maximum value of matrix m.

RETURN VALUE

Maximum value of input matrix

EXAMPLE

#include <mat.h>

...

mat m = mat_new (2,2); 
double mx; 

m[0][0] = 1.; m[0][1] = 3.;
m[1][0] = 2.; m[1][1] = 4.;

mx = mat_max (m);       /* mx = 4. */
