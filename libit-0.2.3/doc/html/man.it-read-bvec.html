it_read_bvec

Name

it_read_bvec -- read a byte vector from a string

Synopsis

#include <it/io.h>

char * it_read_bvec( char * s, bvec * v );

DESCRIPTION

This functions sets an existing binary vector at the address pointed by v to the one described by the string s. This string contains the values of each element of the vector, separated by a space or comma. This list of elements may additionally be enclosed in brackets for increased compatibility with various software.

RETURN VALUE

A pointer on the next position of to be read in the buffer

EXAMPLE

#include <io.h>

...

bvec v = bvec_new(0);
it_read_bvec("[4 254 2 +4, 2 ]", &v); /* v = [ 4 254 2 4 2 ] */
