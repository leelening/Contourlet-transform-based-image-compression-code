mat_apply_function

Name

mat_apply_function -- Apply a function element-wise to input matrix

Synopsis

#include <it/mat.h>

void mat_apply_function( mat m, it_function_t function, it_args_t args );

DESCRIPTION

The function mat_apply_function applies function to every element of matrix m, given parameters args to function. If no argument shall be given to function, set args to NULL.

RETURN VALUE

None
