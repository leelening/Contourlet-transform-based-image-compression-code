it_itransform2D

Name

it_itransform2D -- generic 2D inverse transform

Synopsis

#include <it/transform2D.h>

Mat it_itransform2D( it_transform2D_t *t, Mat coeffs );

DESCRIPTION

This function performs the 2D inverse transform specified by the it_transform2D_t object t on the generic matrix coeffs representing the transform coefficients.

RETURN VALUE

EXAMPLE

#include <transform2D.h>

...

it_wavelet2D_t *wavelet = it_wavelet2D_new(it_wavelet_lifting_97, 5);
mat input = mat_new_ones(16,16);
mat output;
output = (Mat) it_itransform2D(wavelet, (Mat) intput);
