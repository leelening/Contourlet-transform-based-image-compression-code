pnm_info

Name

pnm_info -- Get information from a PNM image file

Synopsis

#include <it/io.h>

int pnm_info( const char * filename, char * p_pnm_type, int * p_width, int * p_height, int * p_max_val, char * comments, int length );

DESCRIPTION

This function reads the width, height, type, maximum value and comments from a PNM image file specified by filename. The values of these informations are stored in the corresponding pointers. The length argument gives the maximum size of the comments buffer in which the comments are stored.

RETURN VALUE

0 if the function was unable to open the file, otherwise 0

EXAMPLE

#include <io.h>

...

int width, height, maxval, type;
char comment[1000];
pnm_info("image.pgm", &type, &width, &height, &maxval, comments, 1000);
