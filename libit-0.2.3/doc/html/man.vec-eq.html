vec_eq

Name

vec_eq -- check if two vectors are equal

Synopsis

#include <it/vec.h>

int vec_eq( vec v1, vec v2 );

DESCRIPTION

The function vec_eq checks if two vectors are equal (component per component).

RETURN VALUE

0 if the two vectors are equal, otherwise a value different from 0
