source_binary

Name

source_binary -- binary source generator

Synopsis

#include <it/source.h>

bvec source_binary( size_t size, double p0 );

DESCRIPTION

This function generates a random binary vector of size bits where the probability of the 0 symbol is given by p0 and the probability of the 1 symbol is given by 1-p0.

RETURN VALUE

EXAMPLE

#include <source.h>

...

bvec v = source_binary(10, 0.5);
