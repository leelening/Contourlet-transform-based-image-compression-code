vec_set_subvector

Name

vec_set_subvector -- set a part of a vector with another vector

Synopsis

#include <it/vec.h>

void vec_set_subvector( vec v, vec s, size_t idx );

DESCRIPTION

The function vec_set_subvector sets the elements of the vector v with the elements of the vector s, starting at position idx.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new_zeros (6);    /* v = [0 0 0 0 0 0]  */
vec s = vec_new_ones (3);     /* s = [1 1 1]        */
vec_set_subvector (v, s, 2);  /* v = [0 0 1 1 1 0]  */
