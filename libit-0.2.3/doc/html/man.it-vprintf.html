it_vprintf

Name

it_vprintf -- formatted output, variable args

Synopsis

#include <it/io.h>

void it_vprintf( char *fmt, va_list args );

DESCRIPTION

The function it_vprintf extends the standard C vprintf function and provides an output according to the format string. Being an extension fprintf it supports all its types and modifiers. See the it_printf function for details on the extensions.

RETURN VALUE

None

EXAMPLE

#include <io.h>

...

void my_it_printf( char *fmt, ... ) {
  va_list ap;
  va_start(ap, fmt);
  it_vprintf(fmt, ap);
  va_end(ap);
}
