imat_incr

Name

imat_incr -- add a value to the elements of a matrix

Synopsis

#include <it/mat.h>

void imat_incr( imat m, int a );

DESCRIPTION

The function imat_incr adds a to all the elements of the matrix m. This function is inplace.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m1 = imat_new_ones (2,2);
imat_incr (m1, 1);               /* m1 = [[2 2]  */
                                          [2 2]  */
