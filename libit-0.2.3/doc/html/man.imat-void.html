imat_void

Name

imat_void -- Set a matrix to void

Synopsis

#include <it/mat.h>

void imat_void( imat m );

DESCRIPTION

The function imat_void sets a matrix to the void matrix, i.e. a matrix containing no element.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m = imat_new_ones(2,2);  /* Matrix of size 2x2                           */
imat_void( m );               /* Matrix does not contains any element anymore */
