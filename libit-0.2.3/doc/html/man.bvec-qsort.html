bvec_qsort

Name

bvec_qsort -- Sort a vector with the quick sort

Synopsis

#include <it/vec.h>

void bvec_qsort( bvec v );

DESCRIPTION

The function bvec_sort sorts a vector with the quick sort algorithm. Note that it makes use of the standard C function qsort.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_string ("1 0 2");   /* v = [1 0 2] */
bvec_qsort (v);                       /* v = [0 1 2] */
