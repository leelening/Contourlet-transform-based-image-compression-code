bvec_set_length_max

Name

bvec_set_length_max -- Modify the amount of memory allocated to a vector

Synopsis

#include <it/vec.h>

void bvec_set_length_max( bvec v, size_t N );

DESCRIPTION

The ivec_set_length_max macro modifies the amount of memory allocated for a vector so that the maximum number of allocated elements is equal to N. It is the equivalent of bvec_new_alloc for existing vectors. This function may modify the pointer representing the vector.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new(10);    /* allocate a new vector of size 10      */
bvec_set_length_max(v,4); /* modify the amount of allocated memory */
bvec_delete(v);           /* free the vector                       */
