mat_add

Name

mat_add -- add a matrix to another

Synopsis

#include <it/mat.h>

void mat_add( mat m1, mat m2 );

DESCRIPTION

The function mat_add adds the matrix m2 to m1 and stores the result in m1. Hence, this function is inplace.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

mat m1 = mat_new_ones (2,2);
mat m2 = mat_new_ones (2,2);
mat_add (m1, m2);               /* m1 = [[2 2]  */
                                         [2 2]  */
