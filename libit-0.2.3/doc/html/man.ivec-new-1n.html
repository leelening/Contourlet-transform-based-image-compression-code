ivec_new_1N

Name

ivec_new_1N -- create a vector of integers 1, 2, ...

Synopsis

#include <it/vec.h>

ivec ivec_new_1N( size_t N );

DESCRIPTION

The function ivec_new_1N creates the vector [1 2 ... N], where the value N denotes the length of the vector.

RETURN VALUE

The new vector

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_1N(5);    /* creates the vector [1 2 3 4 5] */
ivec_delete(v);             /* free the vector                */
