imat_copy_col

Name

imat_copy_col -- Copy the column of a matrix into a vector

Synopsis

#include <it/mat.h>

void imat_copy_col( ivec v, imat m, size_t c );

DESCRIPTION

The function imat_copy_col copies the column c of a matrix m into a vector v.

None
