bmat_new_transpose

Name

bmat_new_transpose -- Return the transpose of a matrix

Synopsis

#include <it/mat.h>

bmat bmat_new_transpose( bmat m );

DESCRIPTION

The function bmat_new_transpose return a matrix which is the transpose of the matrix m, i.e. the matrix such that the rows and the columns are inverted.

RETURN VALUE

The transposed matrix

EXAMPLE

#include <mat.h>

...

bmat m = bmat_new_string ("[[1 2 3][4 5 6]]");  /* m = [[1 2 3]
                                                        [4 5 6]]  */

bmat t = bmat_new_transpose (m);                /* t = [[1 4]
                                                        [2 5]]    */
                                                        [3 6]]    */
