imat_to_ivec

Name

imat_to_ivec -- vectorize a matrix into a vector

Synopsis

#include <it/mat.h>

ivec imat_to_ivec( imat m );

DESCRIPTION

The function imat_to_ivec vectorizes the matrix m into a vector. In other terms, the resulting vector is the concatenation of all the rows of m. Its length is then the product of the width by the height of the matrix 'm '. Note that this function is the equivalent of the notation 'm(:)' in Matlab.

RETURN VALUE

The vector

EXAMPLE

#include <mat.h>

...

imat m = imat_new_string ("[[1 2][4 3]]");  /* m = [[1 2]  
                                                    [4 3]    */

ivec v = imat_to_ivec (m);                   /* v = [1 2 4 3] */
