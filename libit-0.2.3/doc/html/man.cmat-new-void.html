cmat_new_void

Name

cmat_new_void -- Allocate a void matrix

Synopsis

#include <it/mat.h>

cmat cmat_new_void( );

DESCRIPTION

The function cmat_new_void returns a void matrix, i.e. a matrix that contains no elements. The size of the matrix is 1x0.

RETURN VALUE

The matrix

EXAMPLE

#include <mat.h>

...

cmat m = cmat_new_void;   /* m = [[]]  */
...
cmat_delete (m);         /* the matrix has to be free */
