cnorm

Name

cnorm -- modulus of a complex number

Synopsis

#include <it/cplx.h>

double cnorm( cplx a );

DESCRIPTION

The function cnorm returns the modulus of the complex a, i.e. the square root of the sum of the square of the real and imaginary parts.

RETURN VALUE

The module of the complex

EXAMPLE

#include <cplx.h>

...

cplx a = cplx(1,1);
double n = cnorm(a, b);
