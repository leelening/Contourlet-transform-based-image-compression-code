it_uniform_quantizer_new_from_range

Name

it_uniform_quantizer_new_from_range -- create a uniform scalar quantizer from range

Synopsis

#include <it/quantizer.h>

void it_uniform_quantizer_new_from_range( int n, double min, double max );

DESCRIPTION

This function creates a new uniform scalar quantizer object from the range of values to be quantized specified by [min,max] and the number of centroids specified by n. These centroids are equally spaced, with each bin having a step size of (max-min) / n.

RETURN VALUE

EXAMPLE

#include <quantizer.h>

...

/* create a new uniform quantizer with 256 levels
   equally spread in the [0,1] range              */
it_uniform_quantizer_t uniq =
    it_uniform_quantizer_new_from_range(256, 0.0, 1.0);
