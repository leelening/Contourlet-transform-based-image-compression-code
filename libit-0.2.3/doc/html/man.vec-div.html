vec_div

Name

vec_div -- component per component division of 2 vec

Synopsis

#include <it/vec.h>

void vec_div( vec v1, vec v2 );

DESCRIPTION

The function vec_div divides each component of the vector v1 by the corresponding component of the vector v2. Note that the vectors v1 and v2 must have the same length.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

v1 = vec_new_string( "1 0.5 -2 5" );   /* vector [1 0.5 -2 5] */
v2 = vec_new_string( "-3 1 1 5" );     /* vector [-3 1 1 5]   */
vec_div( v1, v2 );                     /* v1 = [0 0 -2 1]     */
vec_delete(v1);
vec_delete(v2);
