imat_transpose

Name

imat_transpose -- Transpose a matrix

Synopsis

#include <it/mat.h>

void imat_transpose( imat m );

DESCRIPTION

The function imat_tranpose tranpose the matrix m. Note that the pointer corresponding to the matrix m may be subsequently modified.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m = imat_new_zeros(5,3); /* Matrix [[0 0 0]
                                         [0 0 0]] */
m[0][1]=1;
m[1][0]=2;
m[1][2]=3;                    /*   m =  [[0 1 0]
                                         [2 0 3]] */
imat_transpose(m);            /*   m =  [[0 2]
                                         [1 0]
                                         [0 3]]   */
