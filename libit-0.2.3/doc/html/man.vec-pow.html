vec_pow

Name

vec_pow -- Set the components of a vector to a power

Synopsis

#include <it/vec.h>

void vec_pow( vec v, double a );

DESCRIPTION

The function vec_pow sets all the components of the vector v to the power a.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string("1 0 3 -2");           /* vector [1 0 3]  */
vec_pow( v, 3 );                              /* v = [1 0 27 -8] */
