vec_clone

Name

vec_clone -- create a copy of a vector

Synopsis

#include <it/vec.h>

vec vec_clone( vec v );

DESCRIPTION

This function creates a copy of the vector v. Note that it is not valid to use the operator = instead.

RETURN VALUE

The new vector, which is a copy of the vector given as a parameter

EXAMPLE

#include <vec.h>

...

vec a = vec_new_string ("0 1 2 3");   /* a = [0 1 2 3] */
vec b = vec_clone (b);                /* b = [0 1 2 3] */
