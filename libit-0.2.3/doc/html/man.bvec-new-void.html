bvec_new_void

Name

bvec_new_void -- create an empty vector

Synopsis

#include <it/vec.h>

bvec bvec_new_void( );

DESCRIPTION

The function bvec_new_void creates a new empty vector, i.e. a vector of length 0.

RETURN VALUE

The void vector

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_void();    /* creates the vector [] */
bvec_delete(v);              /* free the vector       */
