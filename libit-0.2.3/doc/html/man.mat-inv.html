mat_inv

Name

mat_inv -- computes the inverse of the input matrix

Synopsis

#include <it/linalg.h>

mat mat_inv( mat m );

DESCRIPTION

The function mat_inv computes the in-place inverse of the matrix m using LU decomposition. If the matrix is singular, the output is set to NULL, otherwise the output is a pointer to the (modified) input matrix.

RETURN VALUE

NULL is matrix is singular, non-NULL otherwise

EXAMPLE

#include <linalg.h>

...

mat m1 = mat_new( 3, 3 ); 

m1[0][0] = 1;   m1[0][1] = 5;   m1[0][2] = 2; 
m1[1][0] = 2;   m1[1][1] = 3;   m1[1][2] = 5; 
m1[2][0] = 3;   m1[2][1] = 7;   m1[2][2] = 3; 

mat_inv (m1); 

/*
m1=
[[   -0.897    -0.034     0.655]
 [    0.310    -0.103    -0.034]
 [    0.172     0.276    -0.241]]
*/
