cvec_neg

Name

cvec_neg -- Opposite of a vector

Synopsis

#include <it/vec.h>

void cvec_neg( cvec v );

DESCRIPTION

The function vec_neg sets all the components of the vector v to their opposite value.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string("1 1-2i -2+i -i");      /* vector [1 1-2i -2+i -i] */
cvec_neg( v );                                   /* v = [-1 -1+2i 2-i i]    */
