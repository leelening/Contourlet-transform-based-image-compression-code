vec_sum_between

Name

vec_sum_between -- partial sum of a vector

Synopsis

#include <it/vec.h>

double vec_sum_between( vec v, size_t i1, size_t i2 );

DESCRIPTION

The function vec_sum_between returns the sum of the elements of the vector v whose position stands in the interval [i1,i2]. The keyword end may be used for the parameter i2. In that case it means that the summation is calculated from position i1 to the end of the vector.

RETURN VALUE

The partial sum

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string( "5 -2 2 4 1.5" ); /* Create a vector of 5 elements */
double d = vec_sum( v );                  /* d = 10.5                      */
