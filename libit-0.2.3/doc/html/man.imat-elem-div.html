imat_elem_div

Name

imat_elem_div -- Performs element-wise matrix division

Synopsis

#include <it/mat.h>

void imat_elem_div( imat m1, imat m2 );

DESCRIPTION

The function imat_elem_div divides element (i,j) of m1 by element (i,j) of m2 and stores the result in m1.

RETURN VALUE

void
