it_dequantize_mat

Name

it_dequantize_mat -- dequantize a matrix

Synopsis

#include <it/quantizer.h>

mat it_dequantize_mat( it_quantizer_t *q, imat m );

DESCRIPTION

This function dequantizes the matrix m and returns the matrix of reconstructed values. The matrix is processed row by row, using the vector dequantizer provided by q on each row.
