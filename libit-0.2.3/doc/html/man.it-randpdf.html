it_randpdf

Name

it_randpdf -- generic random number generator

Synopsis

#include <it/random.h>

double it_randpdf( double a, double b, it_function_t pdf, it_args_t args );

DESCRIPTION

The function it_randpdf generates a random value drawn from the distribution pdf. This distribution is assumed to be null outside the range [a, b] and to attain its maximal value in 0. Specific parameters for the distribution function are passed through the args argument.

RETURN VALUE

A realisation of the random variable defined by the probability distribution function

EXAMPLE

#include <random.h>

...

/* draw a random laplacian sample */
double s = it_randpdf(-10, 10, laplacian_pdf, &laplacian_args);
