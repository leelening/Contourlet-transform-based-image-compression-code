cmat_delete

Name

cmat_delete -- Free a matrix

Synopsis

#include <it/mat.h>

void cmat_delete( cmat m );

DESCRIPTION

The function cmat_delete frees the memory associated to the matrix m.

RETURN VALUE

void

EXAMPLE

#include <mat.h>

...

cmat m = cmat_new_zeros (3,3);

cmat_delete (m);
