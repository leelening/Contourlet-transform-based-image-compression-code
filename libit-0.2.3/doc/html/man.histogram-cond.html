histogram_cond

Name

histogram_cond -- Process the conditional histogram of a vector

Synopsis

#include <it/source.h>

imat histogram_cond( int omega, ivec S );

DESCRIPTION

The function histogram_cond processes the bi-dimensional histogram of a vector. In other words, it counts the number of occurence of couples in the sequence. This function can be used to estimate the conditional probabilities of an observed sequence.
