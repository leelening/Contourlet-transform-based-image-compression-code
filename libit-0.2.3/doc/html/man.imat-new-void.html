imat_new_void

Name

imat_new_void -- Return a new void matrix

Synopsis

#include <it/mat.h>

imat imat_new_void( );

DESCRIPTION

The function imat_new_void returns a void matrix, i.e. a matrix that contains no elements. The size of the matrix is 1x0.

RETURN VALUE

The matrix

EXAMPLE

#include <mat.h>

...

imat m = imat_new_void;   /* m = [[]]  */
...
imat_delete (m);         /* the matrix has to be free */
