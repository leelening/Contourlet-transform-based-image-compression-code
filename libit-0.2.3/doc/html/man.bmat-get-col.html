bmat_get_col

Name

bmat_get_col -- Get new vector from given column of input matrix

Synopsis

#include <it/mat.h>

bvec bmat_get_col( bmat m, idx_t c );

DESCRIPTION

The function bmat_get_col creates a new vector containing elements of column c of matrix m.

RETURN VALUE

Desired column as a new vector
