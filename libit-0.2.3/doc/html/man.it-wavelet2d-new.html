it_wavelet2D_new

Name

it_wavelet2D_new -- separable 2D wavelet transform constructor

Synopsis

#include <it/wavelet2D.h>

void it_wavelet2D_new( it_wavelet_lifting_t const *lifting, int level );

DESCRIPTION

The it_wavelet2D_new function creates a new separable 2D wavelet transform object on level levels using lifting steps lifting. This object can later be used as an it_transform2D_t to perform discrete wavelet transform or inverse wavelet transform on a real-valued matrix using the it_transform2D or it_itransform2D method respectively.

Creating this transform object and using the it_transform2D method is prefered to using it_dwt2D when multiple transform on matrices of the same size are to be done.

RETURN VALUE

EXAMPLE

#include <wavelet2D.h>

...

it_wavelet_t *wavelet2D = it_wavelet2D_new(it_wavelet_lifting_97, 5);
mat m = mat_new_ones(100,100);
mat mt;
mt = (mat) it_transform2D(wavelet2D, (mat) m);
