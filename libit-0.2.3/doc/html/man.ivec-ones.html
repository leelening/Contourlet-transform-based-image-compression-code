ivec_ones

Name

ivec_ones -- Set all the elements of a vector to one

Synopsis

#include <it/vec.h>

void ivec_ones( ivec v );

DESCRIPTION

The function ivec_ones sets all the elements of an already existing vector to 1.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new(5); /* creates a vector of size 5 (not initialized) */
ivec_ones(v);         /* the vector is set to [1 1 1 1 1]             */
ivec_delete(v);       /* free the vector                              */
