cvec_incr

Name

cvec_incr -- Increment the components of a vector by a value

Synopsis

#include <it/vec.h>

void cvec_incr( cvec v, cplx a );

DESCRIPTION

The function cvec_incr adds the quantity a to all the components of the vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v  = cvec_new_string("0.3-i i 1");  /* vector [ 0.3-i 1+i i ]     */
cplx c = cplx(1,1);
cvec_incr(v, c);                         /* v = [ 1.3 2+2i 1+2i ]      */
cvec_delete(v);
