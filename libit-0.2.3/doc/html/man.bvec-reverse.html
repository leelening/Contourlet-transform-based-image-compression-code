bvec_reverse

Name

bvec_reverse -- Reverse a vector

Synopsis

#include <it/vec.h>

void bvec_reverse( bvec v );

DESCRIPTION

The function bvec_reverse reverse the elements of the vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_string("1 2 3 4");       /* Vector [1 2 3 4] */
bvec_reverse(v);                           /* v = [4 3 2 1]    */
