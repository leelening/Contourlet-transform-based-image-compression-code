parser_get_mat

Name

parser_get_mat -- read a matrix

Synopsis

#include <it/parser.h>

mat parser_get_mat( const parser_t * p, const char * varname );

DESCRIPTION

The function parser_get_mat retrieves a matrix variable of type mat with the parser. The variable varname is the name of the variable to be found.

RETURN VALUE

the matrix read from the sources

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);
mat m        = parser_get_mat (p, "m");
