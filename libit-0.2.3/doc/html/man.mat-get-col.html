mat_get_col

Name

mat_get_col -- Get new vector from given column of input matrix

Synopsis

#include <it/mat.h>

vec mat_get_col( mat m, idx_t c );

DESCRIPTION

The function mat_get_col returns a new vector containing the elements of column number c of matrix m.

RETURN VALUE

Specified column as a vector

EXAMPLE

#include <mat.h>

...

mat m = mat_new (2,2); 
vec c1; 

m[0][0] = 1; m[0][1] = 3;
m[1][0] = 2; m[1][1] = 4;

c1 = mat_get_col (m,1);     /* c1 = [ 1  2 ] */
