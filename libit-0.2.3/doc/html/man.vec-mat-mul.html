vec_mat_mul

Name

vec_mat_mul -- return the product of a vector by a matrix

Synopsis

#include <it/mat.h>

vec vec_mat_mul( vec v, mat m );

DESCRIPTION

The function vec_mat_mul returns the product of the vector v by the matrix m.

RETURN VALUE

The vector

EXAMPLE

#include <mat.h>

...

vec v = vec_new_string ("[1 2]");  /* v = [1 2]     */
mat m = mat_new_ones (2,3);        /* m = [[1 1 1]  
                                           [1 1 1]] */
vec r = vec_mat_mul (v, m);        /* r = [3 3 3]   */
