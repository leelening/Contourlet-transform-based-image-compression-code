cmat_get_row

Name

cmat_get_row -- Get new vector from given row of input matrix

Synopsis

#include <it/mat.h>

cvec cmat_get_row( cmat m, idx_t r );

DESCRIPTION

The function cmat_get_row creates a new vector containing elements of row r of matrix m.

RETURN VALUE

A vector corresponding to the desired row of the matrix
