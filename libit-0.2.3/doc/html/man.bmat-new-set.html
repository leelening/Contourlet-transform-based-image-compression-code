bmat_new_set

Name

bmat_new_set -- new matrix whose elements are set to a given value

Synopsis

#include <it/mat.h>

bmat bmat_new_set( byte val, int h, int w );

DESCRIPTION

The bmat_new_set function is used to create a new matrix of size hxw elements of bytes, where h and w respectively denote the number of rows and columns. All the elements of this matrix are set to the value val.

RETURN VALUE

The new matrix

EXAMPLE

#include <mat.h>

...

bmat m = bmat_new_set (4, 2, 3);   /* m = [[4 4 4]
                                           [4 4 4]  */
