vec_new_ones

Name

vec_new_ones -- create a vector of ones

Synopsis

#include <it/vec.h>

vec vec_new_ones( size_t N );

DESCRIPTION

The function vec_new_ones creates a new vector of N real numbers initialized to the value 1.

RETURN VALUE

The vector of ones

EXAMPLE

#include <vec.h>

...

vec v = vec_new_ones(5);    /* creates the vector [1 1 1 1 1] */
vec_delete(v);              /* free the vector                */
