ivec_incr

Name

ivec_incr -- Increment the components of a vector by a value

Synopsis

#include <it/vec.h>

void ivec_incr( ivec v, int a );

DESCRIPTION

The function ivec_incr adds the quantity a to all the components of the vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string("2 5 -1");   /* vector [ 2 5 -1 ]     */
ivec_incr(v, 2);                      /* v = [ 4 7 1 ]         */
ivec_delete(v);
