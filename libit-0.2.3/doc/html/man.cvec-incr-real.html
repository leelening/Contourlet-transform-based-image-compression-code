cvec_incr_real

Name

cvec_incr_real -- add a value to the real components of a vector

Synopsis

#include <it/vec.h>

void cvec_incr_real( cvec v, double a );

DESCRIPTION

The function cvec_incr_real adds the quantity a to the real part of the components of the vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v  = cvec_new_string("0.3-i i 1");  /* vector [ 0.3-i 1+i i ]     */
cvec_incr_real(v, 1.3);                  /* v = [ 1.6-i 2.3+i 1.3+i ]  */
cvec_delete(v);
