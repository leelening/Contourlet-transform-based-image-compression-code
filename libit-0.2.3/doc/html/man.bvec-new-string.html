bvec_new_string

Name

bvec_new_string -- Create a vector from a string

Synopsis

#include <it/io.h>

bvec bvec_new_string( char * s );

DESCRIPTION

The function bvec_new_string creates a vector from the string s. The string is parsed and uses the following characters as element separators: space and comma. The function allocates some memory.

RETURN VALUE

The vector

EXAMPLE

#include <io.h>

...

bvec v1 = bvec_new_string("0 1 2 3");     /* Vector [0 1 2 3] */
bvec v2 = bvec_new_string("[2,1,2,1]");   /* Vector [2 1 2 1] */
