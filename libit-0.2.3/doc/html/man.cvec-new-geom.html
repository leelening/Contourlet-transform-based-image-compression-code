cvec_new_geom

Name

cvec_new_geom -- create a geometric sequence

Synopsis

#include <it/vec.h>

cvec cvec_new_geom( cplx start, cplx r, size_t N );

DESCRIPTION

The cvec_new_geom function creates a vector generated so that the elements correspond to the geometric sequence of starting value start, generated by the factor of r and of length N.

RETURN VALUE

The new vector

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_geom(1,cplx_I,5); /* creates the vector [1 i -1 -i 1] */ 
cvec_delete(v);                     /* free the vector                  */
