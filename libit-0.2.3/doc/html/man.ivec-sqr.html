ivec_sqr

Name

ivec_sqr -- put the components of a vector to square

Synopsis

#include <it/vec.h>

void ivec_sqr( ivec v );

DESCRIPTION

The function ivec_sqr squares all the components of v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string("[1 2 3 4]");  /* vector [1 2 3 4] */
ivec_sqr( v );                          /* v = [1 4 9 16]    */
