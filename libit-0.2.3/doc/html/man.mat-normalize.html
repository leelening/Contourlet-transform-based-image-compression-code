mat_normalize

Name

mat_normalize -- Normalisation of the matrix (sum=1)

Synopsis

#include <it/mat.h>

void mat_normalize( mat m );

DESCRIPTION

The function mat_normalize normalizes the matrix so that the sum of the element of the matrix is equal to 1. Note that you would prefer to use the function mat_cols_normalize instead.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

mat m = mat_new_ones (2,2);     /* m =[[1 1]
                                       [1 1]]  */

mat_normalize (m);              /* m =[[0.25 0.25]
                                       [0.25 0.25]]  */
