mat_to_vec

Name

mat_to_vec -- vectorize a matrix into a vector

Synopsis

#include <it/mat.h>

vec mat_to_vec( mat m );

DESCRIPTION

The function mat_to_vec vectorizes the matrix m into a vector. In other terms, the resulting vector is the concatenation of all the rows of m. Its length is then the product of the width by the height of the matrix 'm '. Note that this function is the equivalent of the notation 'm(:)' in Matlab.

RETURN VALUE

The vector

EXAMPLE

#include <mat.h>

...

mat m = mat_new_string ("[[1 2][4 3]]");  /* m = [[1 2]  
                                                  [4 3]    */

vec v = mat_to_vec (m);                   /* v = [1 2 4 3] */
