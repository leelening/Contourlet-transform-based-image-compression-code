cvec_set_subvector

Name

cvec_set_subvector -- set a part of a vector with another vector

Synopsis

#include <it/vec.h>

void cvec_set_subvector( cvec v, cvec s, size_t idx );

DESCRIPTION

The function cvec_set_subvector sets the elements of the vector v with the elements of the vector s, starting at position idx.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_zeros (6);    /* v = [0 0 0 0 0 0]  */
cvec s = cvec_new_ones (3);     /* s = [1 1 1]        */
cvec_set_subvector (v, s, 2);   /* v = [0 0 1 1 1 0]  */
