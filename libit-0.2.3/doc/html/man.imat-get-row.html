imat_get_row

Name

imat_get_row -- Get new vector from given row of input matrix

Synopsis

#include <it/mat.h>

ivec imat_get_row( imat m, idx_t r );

DESCRIPTION

The function imat_get_row creates a new vector containing the elements of row r of matrix m.

RETURN VALUE

A vector which is a copy of the desired row
