ivec_new_alloc

Name

ivec_new_alloc -- create a new vector of integers

Synopsis

#include <it/vec.h>

ivec ivec_new_alloc( size_t length, size_t length_max );

DESCRIPTION

The function ivec_new_alloc creates a new vector of length integers represented with the int type. Unlike the function ivec_new, the function ivec_new_alloc allocates memory for a greater number of elements than the vector length. Hence, the number length_max gives the size up to which the vector can grow without any memory reallocation. This function allows to avoid to reallocate memory if the vector comes to grow up to the value length_max. This may be useful if the vector is used as a stack. Note however that in most cases the user does not have to care about the real amount of memory allocated for the vector, since the geometric reallocation strategy allows to avoid to introduce additional complexity while modifying the size of a vector.

RETURN VALUE

The new vector

EXAMPLE

#include <vec.h>

...

int i;
int K = 100;
ivec v = ivec_new_alloc(0, K);/* create a new vector of size 0, and
                                  allocate memory for 100 elements */
for( i = 0 ; i < K ; i++ )    /* Add 100 elements in the vector.   */
  ivec_push(v, i);           
ivec_delete(v);               /* free the vector                   */
