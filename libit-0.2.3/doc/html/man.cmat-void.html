cmat_void

Name

cmat_void -- Set a matrix to void

Synopsis

#include <it/mat.h>

void cmat_void( cmat m );

DESCRIPTION

The function cmat_void sets a matrix to the void matrix, i.e. a matrix containing no element.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

cmat m = cmat_new_ones(2,2);  /* Matrix of size 2x2                           */
cmat_void( m );               /* Matrix does not contains any element anymore */
