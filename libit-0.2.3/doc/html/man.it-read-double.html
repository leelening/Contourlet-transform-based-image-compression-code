it_read_double

Name

it_read_double -- read a real value from a string

Synopsis

#include <it/io.h>

char * it_read_double( char * s, double * p_val );

DESCRIPTION

This functions sets the real value at the address pointed by p_val to the one described by the string s.

RETURN VALUE

A pointer on the next position of to be read

EXAMPLE

#include <io.h>

...

double f;
it_read_double("5.3", &f); /* f = 5.3 */
