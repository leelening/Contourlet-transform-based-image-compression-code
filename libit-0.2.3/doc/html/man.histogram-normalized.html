histogram_normalized

Name

histogram_normalized -- Process the normalized histogram of a source

Synopsis

#include <it/source.h>

vec histogram_normalized( int omega, ivec S );

DESCRIPTION

The function histogram_normalized is very similar to the function histogram. The difference is that it returns a normalized vector of double, which can be seen as the observed probability distribution function defined by S.
