mat_pgm_read

Name

mat_pgm_read -- Read an image (PGM) file as a matrix

Synopsis

#include <it/io.h>

mat mat_pgm_read( const char* filename );

DESCRIPTION

The function mat_pgm_read read the PGM file of type 5 whose name is filename and returns a matrix of double. Since a PGM file store a gray-scaled images, each component of the matrix defines a pixel luminance.

RETURN VALUE

The matrix corresponding to the values of the pixel

EXAMPLE

#include <io.h>

...

mat m = mat_pgm_read("image.pgm");  /* read the image      */
mat_transpose(m);                   /* transpose the image */
mat_pgm_write(m, "image.pgm");      /* write the image     */
