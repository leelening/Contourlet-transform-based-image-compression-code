ivec_set_subvector

Name

ivec_set_subvector -- set a part of a vector with another vector

Synopsis

#include <it/vec.h>

void ivec_set_subvector( ivec v, ivec s, size_t idx );

DESCRIPTION

The function ivec_set_subvector sets the elements of the vector v with the elements of the vector s, starting at position idx.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_zeros (6);    /* v = [0 0 0 0 0 0]  */
ivec s = ivec_new_ones (3);     /* s = [1 1 1]        */
ivec_set_subvector (v, s, 2);   /* v = [0 0 1 1 1 0]  */
