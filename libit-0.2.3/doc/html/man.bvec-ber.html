bvec_ber

Name

bvec_ber -- bit error rate between two sequences

Synopsis

#include <it/distance.h>

double bvec_ber( bvec v1, bvec v2 );

DESCRIPTION

This function computes the bit error rate (BER) between two binary sequences v1 and v2. This corresponds to the Hamming distance between v1 and v2 divided by the length of v1, which is taken as the reference in case the vectors lengths do not match.

RETURN VALUE

The bit error rate

EXAMPLE

#include <distance.h>

...

bvec v1 = bvec_new_string("0 1 1 1 0 0");
bvec v2 = bvec_new_string("0 1 1 0 0 0");
/* compute the BER between v1 and v2 */
double ber = bvec_ber(v1, v2);
