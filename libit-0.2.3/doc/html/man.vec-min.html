vec_min

Name

vec_min -- Minimum of a vector

Synopsis

#include <it/vec.h>

double vec_min( vec v );

DESCRIPTION

The function vec_min returns the minimum of the vector v.

RETURN VALUE

The minimum

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string("1 -2.7 7 0");           /* vector [1 -2.7 7 0] */
double m = vec_min( v );                        /* m = -2.7            */
