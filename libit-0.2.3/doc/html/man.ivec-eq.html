ivec_eq

Name

ivec_eq -- check if two vectors are equal

Synopsis

#include <it/vec.h>

int ivec_eq( ivec v1, ivec v2 );

DESCRIPTION

The function ivec_eq checks if two vectors are equal (component per component).

RETURN VALUE

0 if the two vectors are equal, otherwise a value different from 0
