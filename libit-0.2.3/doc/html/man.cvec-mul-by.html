cvec_mul_by

Name

cvec_mul_by -- multiply a complex vector by a complex value

Synopsis

#include <it/vec.h>

void cvec_mul_by( cvec v, cplx a );

DESCRIPTION

The function cvec_mul_by multiplies each component of the vector v by the quantity a.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string ("1 i 3");   /* v = [1 i 3]   */
cvec_mul_by (v, cplx_I);              /* v = [i -1 3i] */
