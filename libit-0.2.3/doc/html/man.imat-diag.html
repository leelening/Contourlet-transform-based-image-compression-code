imat_diag

Name

imat_diag -- Set a matrix to a diagonal matrix

Synopsis

#include <it/mat.h>

void imat_diag( imat m, ivec v );

DESCRIPTION

The function imat_diag sets a matrix m to a diagonal matrix, i.e. a matrix whose elements are equals to 0 excepted the diagonal which is defined by the vector parameter v

RETURN VALUE

None, as the function is inplace

EXAMPLE

#include <mat.h>

...

imat m = imat_new_ones (2,2); /* m = [[1 1]  */
                              /*      [1 1]  */

ivec d = ivec_new_set (3,2);  /* d = [2 2]   */

imat_diag (m, v);             /* m = [[3 0]  */
                              /*      [0 3]  */
