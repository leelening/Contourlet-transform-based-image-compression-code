mat_new_diag

Name

mat_new_diag -- Creates a diagonal matrix from input vector

Synopsis

#include <it/mat.h>

mat mat_new_diag( vec v );

DESCRIPTION

The function mat_new_diag creates a diagonal matrix from the parameter vector v. The returned matrix is a square matrix whose diagonal correspond to the elements of v. Note that the width and the heigth of the returned matrix are equal to the length of the vector v.

RETURN VALUE

New matrix with element of input vector on the diagonal

EXAMPLE

#include <mat.h>

...

vec v = vec_new (2); 
mat m; 

v[0] = 2; v[1] = 3; 

m = mat_new_diag (v); 

/*
m = 
[[ 2   0 ]
 [ 0   3 ]]
*/
