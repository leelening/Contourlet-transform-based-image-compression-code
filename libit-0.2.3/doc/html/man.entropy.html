entropy

Name

entropy -- first-order entropy rate of a discrete source

Synopsis

#include <it/source_func.h>

double entropy( vec pdf );

DESCRIPTION

This function returns the entropy rate, i.e. the minimum achievable average description length of a discrete first-order source described by pdf. This vector gives the probability of each symbol.

RETURN VALUE

EXAMPLE

#include <source_func.h>

...

vec pdf = vec_new_string("0.5 0.3 0.2");
double H = entropy(pdf);
