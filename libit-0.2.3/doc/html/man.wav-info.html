wav_info

Name

wav_info -- Get information from a WAV sound file

Synopsis

#include <it/io.h>

int wav_info( const char * filename, int * p_channels, int *p_srate, int *p_depth, int *p_length );

DESCRIPTION

This function reads the length, sampling rate, number of channels and sample bit depth from a WAV sound file specified by filename. The values of these informations are stored in the corresponding pointers p_length, p_srate, p_channels and p_depth respectively.

RETURN VALUE

1 if the function achieved to open the file, otherwise 0

EXAMPLE

#include <io.h>

...

int length, srate, channels, depth;
wav_info("sound.wav", &channels, &srate, &depth, &length);
