vec_mean

Name

vec_mean -- mean value of a vector

Synopsis

#include <it/vec.h>

double vec_mean( vec v );

DESCRIPTION

The function vec_mean computes the arithmetic mean of vector v.

RETURN VALUE

arithmetic mean of input vector

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string( "5 -2 2 4 1" );  /* Create a vector of 5 elements */
double d = vec_mean( v );                 /* d = 2                        */
