parser_get_cplx

Name

parser_get_cplx -- read a complex

Synopsis

#include <it/parser.h>

cplx parser_get_cplx( const parser_t * p, const char * varname );

DESCRIPTION

The function parser_get_cplx retrieves a complex variable of type cplx with the parser. The variable varname is the name of the variable to be found.

RETURN VALUE

the complex read from the source of the parser

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);
cplx         = parser_get_cplx (p, "mycplx");
