bmat_set_height

Name

bmat_set_height -- set the height of a matrix

Synopsis

#include <it/mat.h>

void bmat_set_height( bmat m, size_t h );

DESCRIPTION

This function sets the height of the matrix m to height.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

cmat m = cmat_new(5, 5);
cmat_set_height(m, 10); /* m is now a 10-row 5-column matrix */
