ivec_inner_product

Name

ivec_inner_product -- Inner product of two vectors

Synopsis

#include <it/vec.h>

int ivec_inner_product( ivec v1, ivec v2 );

DESCRIPTION

This function returns the inner product of the vectors v1 and v2. These vectors must be of same length.

RETURN VALUE

The inner product

EXAMPLE

#include <vec.h>

...

ivec v1 = ivec_new_string("-1 -2 1.5");     /* vector [ -1 -2 1.5 ]  */
ivec v2 = ivec_new_string("2 0 4");         /* vector [ 2 0 4 ]      */
int p = vec_inner_product(v1, v2);          /* p = 4                 */
