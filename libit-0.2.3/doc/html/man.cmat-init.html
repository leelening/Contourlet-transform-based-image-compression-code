cmat_init

Name

cmat_init -- init a matrix from a buffer

Synopsis

#include <it/mat.h>

void cmat_init( cmat m, cplx * buf, size_t w, size_t h );

DESCRIPTION

The function cmat_init copies the elements of the buffer into the matrix, row per row.

RETURN VALUE

None
