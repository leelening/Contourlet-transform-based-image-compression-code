cmat_eq

Name

cmat_eq -- Test matrices for equality

Synopsis

#include <it/mat.h>

int cmat_eq( cmat m1, cmat m2 );

DESCRIPTION

The function cmat_eq tests if m1 and m2 are equal and return a non-zero integer if m1==m2, or zero otherwise.

RETURN VALUE

Zero if 'm1'!='m2', or a non-zero integer otherwise

EXAMPLE

#include <mat.h>

...

cmat m1, m2;
m1 = cmat_new_zeros(2,2);
m2 = cmat_new_ones(2,2);
if(cmat_eq(m1, m2))
  printf("the matrices are equal");
else
  printf("the matrices are not equal");
