parser_get_ivec

Name

parser_get_ivec -- read a vector

Synopsis

#include <it/parser.h>

ivec parser_get_ivec( const parser_t * p, const char * varname );

DESCRIPTION

The function parser_get_ivec retrieves a vector of type ivec with the parser. The variable varname is the name of the variable to be found.

RETURN VALUE

The vector of integers read by the parser

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);
vec v        = parser_get_vec (p, "myvec");
