imat_wav_read

Name

imat_wav_read -- Read a wav file into a multi-channel matrix

Synopsis

#include <it/io.h>

imat imat_wav_read( const char* filename );

DESCRIPTION

The function imat_wav_read reads a WAV file named filename and returns a matrix of integers. Each row of this matrix corresponds to a channel in the WAV file. For example, a mono WAV file will result in a matrix with one row, whereas a stereo WAV file will result in a matrix with two rows. Each row is a vector representing the samples of the sound for each instant. The number of bits used to represent each sample and sampling frequency depend on the actual WAV file and can be retrieved via wav_info.

RETURN VALUE

The matrix containing a row per channel

EXAMPLE

#include <io.h>

...

imat m;
m = imat_wav_read("sound.wav");
