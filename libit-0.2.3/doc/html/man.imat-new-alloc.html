imat_new_alloc

Name

imat_new_alloc -- create a matrix and specify the maximum dimensions

Synopsis

#include <it/mat.h>

imat imat_new_alloc( size_t h, size_t w, size_t hmax, size_t wmax );

DESCRIPTION

The function imat_new_alloc creates a matrix of size hxw, but allocate the memory for a matrix of size hmaxxwmax. This will allow the matrix to grow up to this size without any re-allocation of the matrix.

RETURN VALUE

The new matrix

EXAMPLE

#include <mat.h>

...

imat m = imat_new_alloc (1,2,3,4);   /* m of size 1x2  */

int h  = imat_height (m);            /* h = 1          */
int w  = imat_width (m);             /* w = 2          */
int hm = imat_height_max (m);        /* hm = 3         */
int wm = imat_width_max (m);         /* wm = 4         */
