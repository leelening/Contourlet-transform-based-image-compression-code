cvec_arithm

Name

cvec_arithm -- Define a arithmetic sequence within a vector

Synopsis

#include <it/vec.h>

void cvec_arithm( cvec, cplx start, cplx incr );

DESCRIPTION

The function cvec_arithm sets the vector v so that the elements correspond to the arithmetic sequence of starting value start, increasing by a step of incr. The length is kept unchanged.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new(4);     /* creates the vector (not initialized) */
cvec_arithm(v,1,cplx_I);  /* set v to [1 1+i 1+2i 1+3i]           */
