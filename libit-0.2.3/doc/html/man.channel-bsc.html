channel_bsc

Name

channel_bsc -- BSC channel simulation

Synopsis

#include <it/channel.h>

bvec channel_bsc( bvec v, double crossover_proba );

DESCRIPTION

This function simulates the transmission of a binary vector v over an binary symmetric channel (BSC). The cross-over probability of this channel is given by the crossover_proba argument.

RETURN VALUE

The noisy sequence of bits

EXAMPLE

#include <channel.h>

...

bvec message = bvec_new_string("0 1 1 0 0 0 1");

/* send over a BSC channel of cross-over probability 0.1 */ 
bvec received = channel_bsc(message, 0.1);
