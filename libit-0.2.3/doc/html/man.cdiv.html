cdiv

Name

cdiv -- Division of two complex numbers

Synopsis

#include <it/cplx.h>

cplx cdiv( cplx a, cplx b );

DESCRIPTION

The function cdiv divides the complex number a by the complex number b

RETURN VALUE

The result of the division

EXAMPLE

#include <cplx.h>

...

cplx a = cplx(1,1);
cplx b = cplx(2,3);
cplx c = cdiv(a, b);
