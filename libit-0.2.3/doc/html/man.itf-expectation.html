itf_expectation

Name

itf_expectation -- expectation operator

Synopsis

#include <it/math.h>

double itf_expectation( double x, it_function_args(itf_expectation) *args );

DESCRIPTION

The function itf_expectation returns the expectation of the an it_function f defined as the integral of t*f(t) over the interval. This is also sometimes called the first moment of the function f. The argument x corresponds to the upper bound of this integral. The expectation of the function is approximated by computing the value of the function at various positions between a and x and using the Romberg method to compute a fast and accurate approximation. The expectation operator has the following parameters, passed through the args argument.

Function parameters

a

The lower bound of the integral.

function

The function on which to compute the expectation.

args

The parameters of the function.

RETURN VALUE

Not really applicable. See the example below.

EXAMPLE

#include <math.h>

...

/* declare the expectation parameters */
it_function_args(itf_expectation) expectation_args;

/* compute the expectation of the identity function,  */
/* limited to the range [0,sqrt(2)]                   */
/* (this defines a valid p.d.f.)                      */
expectation_args.function = itf_identity;
expectation_args.args = NULL;
expectation_args.a = 0.0;
itf_integrate_romberg(sqrt(2.0), &expectation_args);
