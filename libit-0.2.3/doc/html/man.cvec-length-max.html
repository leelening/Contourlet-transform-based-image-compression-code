cvec_length_max

Name

cvec_length_max -- get the number of elements allocated to a vector

Synopsis

#include <it/vec.h>

int cvec_length_max( cvec v );

DESCRIPTION

The function cvec_length returns the maximum length of the vector v. By maximum length, it does not mean that the vector can't grow. It just means the length to which the vector can grow without a call (implicit or explicit) to function cvec_set_length_mat.

RETURN VALUE

The maximum length

EXAMPLE

#include <vec.h>

...

cvec v  = cvec_new_alloc (1,3);
v[0]   = 3;                      /* v = [3]   */
int l  = cvec_length (v);        /* l = 1     */
int l  = cvec_length_max(v);     /* l = 3     */

cvec_push (v, 2);                /* v = [3 2], and there is not re-allocation */
