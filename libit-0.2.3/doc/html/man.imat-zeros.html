imat_zeros

Name

imat_zeros -- Set the elements of a matrix to 0

Synopsis

#include <it/mat.h>

void imat_zeros( imat m );

DESCRIPTION

The function mat_zeros sets all the elements of the matrix m to 0. Hence, the size of the matrix m is not modified.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m = imat_new_ones (2,2); /* m = [[1 1]  */
                                      [1 1]  */

imat_zeros (m);               /* m = [[0 0]  */
                                      [0 0]  */
