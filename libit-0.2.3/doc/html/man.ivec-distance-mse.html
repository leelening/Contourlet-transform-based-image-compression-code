ivec_distance_mse

Name

ivec_distance_mse -- mean square error between two sequences

Synopsis

#include <it/distance.h>

double ivec_distance_mse( ivec v1, ivec v2, double rec_value );

DESCRIPTION

This function computes the mean square error (MSE) between two sequences v1 and v2. This error is defined as the average of the square of the difference between each element of v1 and the corresponding element of v2. If the vectors are not of the same length, the shortest vector is padded with the value rec_value from its end up to the length of the longest vector before computing the MSE.

RETURN VALUE

The mean square error

EXAMPLE

#include <distance.h>

...

ivec v1 = ivec_new_string("0 1 0 3");
ivec v2 = ivec_new_string("1 5 4 6");
/* compute the MSE between v1 and v2 */
double mse = ivec_distance_mse(v1, v2, 0);
