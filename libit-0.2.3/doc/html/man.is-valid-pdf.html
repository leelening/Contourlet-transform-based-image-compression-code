is_valid_pdf

Name

is_valid_pdf -- Check if a vector defines a valid pdf

Synopsis

#include <it/source.h>

int is_valid_pdf( vec pdf, double tol );

DESCRIPTION

The function is_valid_pdf verifies that the summation of the vector pdf, seen as the probability distribution function over a finite set, is equal to 1 with the tolerance tol.
