imat_set_row

Name

imat_set_row -- modify a row of a matrix with a vector

Synopsis

#include <it/mat.h>

void imat_set_row( imat m, int row, int value );

DESCRIPTION

This macro sets an entire row row of the matrix m to the value value.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m = imat_new_zeros(2,2);
ivec v = ivec_new_ones (2);
imat_set_row(m, 1, v);       /* [[0 1]
                                  [0 1]] */
