it_dft

Name

it_dft -- discrete Fourier transform

Synopsis

#include <it/fourier.h>

cvec it_dft( cvec v );

DESCRIPTION

This function computes the discrete Fourier transform of the complex vector v. The first element of the returned vector corresponds to the lowest frequency (DC value). For a complex vector v that is actually a real vector (i.e. when all elements of v have a null imaginary part), the output vector will be conjugate symmetric, that is V[N-i] = V[i]*, where V is the output vector of length N equal to the length of the input vector v.

RETURN VALUE

The vector of discrete Fourier coefficients

EXAMPLE

#include <fourier.h>

...

cvec v = cvec_new_string("1.0 4.0 1.0 -2.0");
/* analyse the vector using the discrete Fourier transform */
cvec vt = it_dft(v);
