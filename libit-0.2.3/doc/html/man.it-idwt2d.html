it_idwt2D

Name

it_idwt2D -- inverse separable 2D discrete wavelet transform

Synopsis

#include <it/wavelet2D.h>

mat it_idwt2D( mat t, it_wavelet_lifting_t const *lifting, int levels );

DESCRIPTION

This function computes the inverse discrete wavelet transform of mt using the lifting steps described by lifting. The input matrix of coefficients mt is the concatenation of all the coefficients in each subbands starting from the low-frequency subband. Currently the lifting steps for the usual 5/3 and 9/7 biorthogonal filters are predefined as "it_wavelet_lifting_53" and "it_wavelet_lifting_97", although you can provide your own lifting steps.

RETURN VALUE

The set of inverse wavelet coefficients

EXAMPLE

#include <wavelet2D.h>

...

/* analyse a signal 'm' using a 5-level 9/7 wavelet decomposition */
mat mt = it_dwt2D(m, it_wavelet_lifting_97, 5);
/* recompose the signal by inverse transform */
mat mr = it_idwt2D(mt, it_wavelet_lifting_97, 5);
