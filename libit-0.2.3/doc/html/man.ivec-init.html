ivec_init

Name

ivec_init -- Initialization of a vector from a buffer of int

Synopsis

#include <it/vec.h>

void ivec_init( ivec v, int * buf, size_t N );

DESCRIPTION

The ivec_init function copies the N first elements of the memory area pointed by pointer buf. into vector v. Note that the vector v must be allocated at the time of the call.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_zeros(15);/* allocate a new vector of zeros of size 15 */
int * buf = (int *) malloc( 13 * sizeof(int) );
ivec_init(v,buf,13);       /* Copy the start the buffer into the vector */
ivec_delete(v);            /* free the vector and the buffer */
free( buf );
