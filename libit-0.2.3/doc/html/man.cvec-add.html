cvec_add

Name

cvec_add -- add a vector to another

Synopsis

#include <it/vec.h>

void cvec_add( cvec v1, cvec v2 );

DESCRIPTION

The function adds the vector v2 to the vector v1. The result is stored in the vector v1. Note that the vectors v1 and v2 must have the same length.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v1 = cvec_new_string( "i 0.5 -2 5" );   /* vector [i 0.5 -2 5] */
cvec v2 = cvec_new_string( "-3 0 i 5" );     /* vector [-3 0 i 5]   */
cvec_add( v1, v2 );                          /* v1 = [i-3 0.5 i-2 10] */
cvec_delete(v1);
cvec_delete(v2);
