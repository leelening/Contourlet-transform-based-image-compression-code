vec_exp

Name

vec_exp -- Exponentation of the components

Synopsis

#include <it/vec.h>

void vec_exp( vec v );

DESCRIPTION

The function vec_exp apply the exponential function to all the components of v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string("0 1 2");  /* vector [ 0 1 2 ]           */
vec_exp( v );                     /* v = [1.0000 2.7183 7.3891] */
