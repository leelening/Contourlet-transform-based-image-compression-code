cmat_height

Name

cmat_height -- Get the height of a matrix

Synopsis

#include <it/mat.h>

int cmat_height( cmat m );

DESCRIPTION

This macro returns the height of the matrix m.

RETURN VALUE

Height of input matrix

EXAMPLE

#include <mat.h>

...

cmat m = cmat_new(5, 7);
int h  = cmat_height(m); /* h = 5 */
