vec_delete

Name

vec_delete -- free a vector

Synopsis

#include <it/vec.h>

void vec_delete( vec v );

DESCRIPTION

The function vec_delete free a previously allocated vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new(10);   /* allocate a new vector of size 10 */
vec_delete(v);         /* free the vector */
