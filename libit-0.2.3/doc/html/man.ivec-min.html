ivec_min

Name

ivec_min -- Minimum of a vector

Synopsis

#include <it/vec.h>

int ivec_min( ivec v );

DESCRIPTION

The function ivec_min returns the minimum of the vector v.

RETURN VALUE

The minimum

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string("1 -2 7 0");          /* vector [1 -2 7 0] */
int m  = ivec_min( v );                        /* m = -2            */
