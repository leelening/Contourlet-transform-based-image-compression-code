parser_delete

Name

parser_delete -- free a parser object

Synopsis

#include <it/parser.h>

void parser_delete( parser_t * p );

DESCRIPTION

The function parser_delete free a parser that has been previously allocated with the function parser_new.

RETURN VALUE

None

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);

...

parser_delete (p);
