imat_min

Name

imat_min -- Return the minimum value of input matrix

Synopsis

#include <it/mat.h>

int imat_min( imat m );

DESCRIPTION

The function imat_min returns the minimum value of matrix m.

RETURN VALUE

Minimum value of input matrix

EXAMPLE

#include <mat.h>

...

imat m = imat_new_string ("[[2 1][-1 5]]");
int min = imat_min (m);                      /* min = -1 */
