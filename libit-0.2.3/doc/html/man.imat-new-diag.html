imat_new_diag

Name

imat_new_diag -- Creates a diagonal matrix from input vector

Synopsis

#include <it/mat.h>

imat imat_new_diag( ivec v );

DESCRIPTION

The function imat_new_diag creates a diagonal matrix from the parameter vector v. The returned matrix is a square matrix whose diagonal correspond to the elements of v. Note that the width and the heigth of the returned matrix are equal to the length of the vector v.

RETURN VALUE

New matrix with element of input vector on the diagonal

EXAMPLE

#include <mat.h>

...

ivec v = ivec_new_string("1 2 3 4 5");   /* Vector [1 2 3 4 5 ]  */
imat m = imat_new_diag(v);               /* Matrix [[1 0 0 0 0]
                                                    [0 2 0 0 0]
                                                    [0 0 3 0 0]
                                                    [0 0 0 4 0]
                                                    [0 0 0 0 5]] */
