mat_norm_inf

Name

mat_norm_inf -- return the infinity norm of a given matrix

Synopsis

#include <it/mat.h>

double mat_norm_inf( mat m );

DESCRIPTION

The function mat_norm_inf returns the infinity norm of the matrix m

RETURN VALUE

The value of the infinity norm
