parser_get_bvec

Name

parser_get_bvec -- read a vector of bytes

Synopsis

#include <it/parser.h>

bvec parser_get_bvec( const parser_t * p, const char * varname );

DESCRIPTION

The function parser_get_bvec retrieves a vector of type 'bvec with the parser. The variable varname is the name of the variable to be found.

RETURN VALUE

the vector

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);
bvec bv      = parser_get_bvec (p, "myvectorofbyte");
