mat_clone

Name

mat_clone -- Creates a new matrix with elements of input matrix

Synopsis

#include <it/mat.h>

mat mat_clone( mat m );

DESCRIPTION

The function mat_clone creates a new matrix with same dimensions as m and copies elements of m in the newly created matrix.

RETURN VALUE

Cloned matrix

EXAMPLE

#include <mat.h>

...

mat m1 = mat_new_zeros (2,2); 
mat m2 = mat_clone (2,2); 

/*
m2 = 
[[ 0  0 ]
 [ 0  0 ]]
*/
