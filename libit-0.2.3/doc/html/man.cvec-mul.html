cvec_mul

Name

cvec_mul -- component per component multiplication of 2 cvec

Synopsis

#include <it/vec.h>

cvec cvec_mul( cvec v1, cvec v2 );

DESCRIPTION

The function cvec_mul multiplies each component of the vector v1 by the corresponding component of the vector v2. Note that the vectors v1 and v2 must have the same length.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

v1 = cvec_new_string( "1 i -2 5" );     /* vector [1 i -2 5] */
v2 = cvec_new_string( "-3 0 1 5" );     /* vector [-3 0 1 5]   */
cvec_mul( v1, v2 );                     /* v1 = [-3 0 -2 25]   */
cvec_delete(v1);
cvec_delete(v2);
