cvec_init

Name

cvec_init -- Initialization of a vector from a buffer

Synopsis

#include <it/vec.h>

void cvec_init( cvec v, cplx * buf, size_t N );

DESCRIPTION

The cvec_init function copies the N first elements of the memory area pointed by pointer buf. into vector v. Note that the vector v must be allocated at the time of the call.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_zeros(15);/* allocate a new vector of zeros of size 15 */
cplx * buf = (cplx *) malloc( 13 * sizeof(cplx) );
cvec_init(v,buf,13);       /* Copy the start the buffer into the vector */
cvec_delete(v);            /* free the vector and the buffer */
free( buf );
