ivec_del

Name

ivec_del -- Delete an element of a vector

Synopsis

#include <it/vec.h>

void ivec_del( ivec v, size_t pos );

DESCRIPTION

The function ivec_del deletes the element at position pos in the vector v. The resulting vector has a length which is equal to the original length minus 1. Note that positions start at 0.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_string ("5 0 6 7 2 1");  /* v = [5 0 6 7 2 1] */
ivec_del (v, 0);                           /* v = [0 6 7 2 1]   */
ivec_del (v, 3);                           /* v = [0 6 7 1]      */
