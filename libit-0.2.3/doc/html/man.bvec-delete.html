bvec_delete

Name

bvec_delete -- free a vector of bytes

Synopsis

#include <it/vec.h>

void bvec_delete( bvec v );

DESCRIPTION

The function bvec_delete free a previously allocated vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new(10);   /* allocate a new vector of size 10 */
bvec_delete(v);          /* free the vector                  */
