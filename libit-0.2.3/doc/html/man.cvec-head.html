cvec_head

Name

cvec_head -- retrieve the head of a vector

Synopsis

#include <it/vec.h>

cplx cvec_head( const cvec v );

DESCRIPTION

Since the vector is seen as a stack, the head is the last element of the vector, i.e. 'v[vec_length(v) - 1]';

RETURN VALUE
