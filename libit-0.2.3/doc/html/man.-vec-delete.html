Vec_delete

Name

Vec_delete -- free a vector

Synopsis

#include <it/vec.h>

void Vec_delete( Vec v );

DESCRIPTION

The Vec_delete macro frees a previously allocated vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

Vec v = Vec_new(float, 10);   /* allocate a new vector of size 10 */
Vec_delete(v);         /* free the vector */
