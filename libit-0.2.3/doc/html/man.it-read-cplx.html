it_read_cplx

Name

it_read_cplx -- read a complex from a string

Synopsis

#include <it/io.h>

char * it_read_cplx( char * s, cplx * p_val );

DESCRIPTION

This functions sets the complex at the address pointed by p_val to the one described by the string s.

RETURN VALUE

A pointer on the next position of to be read in the buffer

EXAMPLE

#include <io.h>

...

cplx z;
it_read_cplx("5.3+4.6*i", &z); /* z = 5.3 + 4.6 * i */
