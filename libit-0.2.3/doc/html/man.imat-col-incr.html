imat_col_incr

Name

imat_col_incr -- add a value to the elements of a given column

Synopsis

#include <it/mat.h>

void imat_col_incr( imat m, size_t col, int a );

DESCRIPTION

The function imat_col_incr adds a to all the elements of the column col in the matrix m

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m = imat_new_set (3,2,2);       /* m = [[3 3]
                                             [3 3]]    */

imat_col_incr (m, 1);                /* m = [[3 4]
                                             [3 4]]    */
