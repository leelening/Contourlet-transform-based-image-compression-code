imat_distance_mse

Name

imat_distance_mse -- mean square error between two images

Synopsis

#include <it/distance.h>

double imat_distance_mse( imat m1, imat m2, double rec_value );

DESCRIPTION

This function computes the mean square error (MSE) between two matrices m1 and m2. This error is defined as the average of the square of the difference between each element of v1 and the corresponding element of v2. If the matrices are not of the same size, the missing elements are assumed to be equal to the value rec_value when computing the MSE.

RETURN VALUE

The mean square error

EXAMPLE

#include <distance.h>

...

imat m1 = imat_new(2,2);
imat m2 = imat_new(2,2);
m1[0][0] = 3;
m1[0][1] = 6;
m1[1][0] = 2;
m1[1][1] = 5;
m2[0][0] = 5;
m2[0][1] = 2;
m2[1][0] = 7;
m2[1][1] = 0;
/* compute the MSE between v1 and v2 */
double mse = imat_distance_mse(v1, v2, 0);
