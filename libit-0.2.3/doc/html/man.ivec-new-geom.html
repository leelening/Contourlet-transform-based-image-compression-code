ivec_new_geom

Name

ivec_new_geom -- create a geometric sequence

Synopsis

#include <it/vec.h>

ivec ivec_new_geom( int start, int r, size_t N );

DESCRIPTION

The function ivec_new_geom creates a vector generated so that the elements correspond to the geometric sequence of starting value start, generated by the factor of r and of length N.

RETURN VALUE

The new vector

EXAMPLE

#include <vec.h>

...

ivec v = ivec_new_geom(1,2,6); /* creates the vector [1 2 4 8 16 32] */ 
ivec_delete(v);                /* free the vector                    */
