mat_elem_sub

Name

mat_elem_sub -- Performs element-wise matrix substraction

Synopsis

#include <it/mat.h>

void mat_elem_sub( mat m1, mat m2 );

DESCRIPTION

The function mat_elem_sub substracts element (i,j) of m2 from element (i,j) of m1 and stores the result in m1.

RETURN VALUE

void

EXAMPLE

#include <mat.h>

...

mat m1 = mat_new(2,2);
mat m2; 

m1[0][0] = 1.; m1[0][1] = 3.;
m1[1][0] = 2.; m1[1][1] = 4.; 

m2 = mat_clone (m1); 

mat_elem_div (m1,m2); 

/*
m1 = 
[[ 0   0 ]
 [ 0   0 ]]
*/
