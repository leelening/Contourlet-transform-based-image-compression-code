ivec_ser

Name

ivec_ser -- symbol error rate between two sequences

Synopsis

#include <it/distance.h>

double ivec_ser( ivec v1, ivec v2 );

DESCRIPTION

This function computes the symbol error rate (SER) between two sequences v1 and v2. This corresponds to the Hamming distance between v1 and v2 divided by the length of v1, which is taken as the reference in case the vectors lengths do not match.

RETURN VALUE

The symbol Error Rate

EXAMPLE

#include <distance.h>

...

ivec v1 = ivec_new_string("1 5 6 6 4");
ivec v2 = ivec_new_string("2 4 5 6 7");
/* compute the SER between v1 and v2 */
double ser = ivec_ser(v1, v2);
