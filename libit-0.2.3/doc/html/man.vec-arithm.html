vec_arithm

Name

vec_arithm -- set the vector to an arithmetic sequence

Synopsis

#include <it/vec.h>

void vec_arithm( vec v, double start, double incr );

DESCRIPTION

The function vec_arithm sets the vector v so that the elements correspond to the arithmetic sequence of starting value start, increasing by a step of incr. The length is kept unchanged.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new(5);    /* creates the vector (not initialized) */
vec_arithm(v,5,-2.5);  /* set v to [5 2.5 0 -2.5 -5]           */
vec_delete(v);         /* free the vector                      */
