cvec_cum_sum

Name

cvec_cum_sum -- Cumulative sum of a vector

Synopsis

#include <it/vec.h>

cvec cvec_cum_sum( cvec v );

DESCRIPTION

The function cvec_cum_sum returns the cumulative sum of the vector v. The cumulative sum is a vector defined so that the component i of this vector represent the summation of the element of v from position 0 to position i (included).

RETURN VALUE

The vector of partial summations

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string( "i -2 2 4 -2i" );  /* Create a vector of 5 elements */
cvec s = cvec_cum_sum( v );                  /* s = [i -2+i i 4+i 4-i ]       */
