imat_add

Name

imat_add -- add a matrix to another

Synopsis

#include <it/mat.h>

void imat_add( imat m1, imat m2 );

DESCRIPTION

The function imat_add adds the matrix m2 to m1 and stores the result in m1. Hence, this function is inplace.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m1 = imat_new_ones (2,2);
imat m2 = imat_new_ones (2,2);
imat_add (m1, m2);               /* m1 = [[2 2]  */
                                          [2 2]  */
