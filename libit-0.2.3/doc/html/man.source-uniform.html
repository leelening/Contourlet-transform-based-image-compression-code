source_uniform

Name

source_uniform -- uniform source generator

Synopsis

#include <it/source.h>

vec source_uniform( size_t size, double a, double b );

DESCRIPTION

This function generates a random continuous vector of length size where each sample is drawn in the range [a, b] from a uniform distribution.

RETURN VALUE

EXAMPLE

#include <source.h>

...

vec v = source_uniform(10, 0.0, 1.0);
