bvec_void

Name

bvec_void -- set a vector to the void vector

Synopsis

#include <it/vec.h>

void bvec_void( bvec v );

DESCRIPTION

The function bvec_void set the vector length of v to 0. The memory allocated to this vector remains allocated.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

v = bvec_new_ones(5);
bvec_void(v);            /* Set the vector to []  */
bvec_delete(v);          /* free the vector       */
