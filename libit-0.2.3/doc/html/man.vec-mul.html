vec_mul

Name

vec_mul -- component per component multiplication of 2 vec

Synopsis

#include <it/vec.h>

void vec_mul( vec v1, vec v2 );

DESCRIPTION

The function vec_mul multiplies each component of the vector v1 by the corresponding component of the vector v2. Note that the vectors v1 and v2 must have the same length.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

v1 = vec_new_string( "1 0.5 -2 5" );   /* vector [1 0.5 -2 5] */
v2 = vec_new_string( "-3 0 1 5" );     /* vector [-3 0 1 5]   */
vec_mul( v1, v2 );                     /* v1 = [-3 0 -2 25]   */
vec_delete(v1);
vec_delete(v2);
