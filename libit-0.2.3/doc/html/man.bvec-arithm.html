bvec_arithm

Name

bvec_arithm -- set the vector to an arithmetic sequence

Synopsis

#include <it/vec.h>

void bvec_arithm( bvec v byte start, byte incr );

DESCRIPTION

The function bvec_arithm sets the vector v so that the elements correspond to the arithmetic sequence of starting value start, increasing by a step of incr. The length is kept unchanged.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new(5); /* creates the vector (not initialized) */
bvec_arithm(v,1,2);   /* set v to [1 3 5 7 9]                 */
bvec_delete(v);       /* free the vector                      */
