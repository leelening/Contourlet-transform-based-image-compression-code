imat_elem_sub

Name

imat_elem_sub -- Performs element-wise matrix substraction

Synopsis

#include <it/mat.h>

void imat_elem_sub( imat m1, imat m2 );

DESCRIPTION

The function imat_elem_sub substracts element (i,j) of m2 from element (i,j) of m1 and stores the result in m1.

RETURN VALUE

void
