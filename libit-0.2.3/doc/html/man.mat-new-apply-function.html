mat_new_apply_function

Name

mat_new_apply_function -- matrix from element-wise function to input matrix

Synopsis

#include <it/mat.h>

mat mat_new_apply_function( mat v, it_function_t function, it_args_t args );

DESCRIPTION

The function mat_new_apply_function applies function to every element of matrix m, given parameters args to function. If no argument shall be given to function, set args to NULL.

RETURN VALUE

The new matrix
