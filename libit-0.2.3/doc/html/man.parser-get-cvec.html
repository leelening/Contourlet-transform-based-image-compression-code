parser_get_cvec

Name

parser_get_cvec -- read a complex vector

Synopsis

#include <it/parser.h>

cvec parser_get_cvec( const parser_t * p, const char * varname );

DESCRIPTION

The function parser_get_cvec retrieves a vector of type cvec with the parser. The variable varname is the name of the variable to be found.

RETURN VALUE

the vector of complex read from the sources

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);
cvec cv      = parser_get_cvec (p, "cv");
