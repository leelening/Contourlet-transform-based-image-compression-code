vec_distance_norm

Name

vec_distance_norm -- norm distance between two vectors

Synopsis

#include <it/distance.h>

double vec_distance_norm( vec v1, vec v2, double norm );

DESCRIPTION

This function computes the distance between the two vectors v1 and v2 corresponding to the norm norm. This norm is defined as the sum of the absolute differences between elements of v1 and the corresponding elements of v2 raised to the power of norm. The norm argument is usually set to 2, which corresponds to the classical Euclidean norm.

RETURN VALUE

The distance derived from the norm

EXAMPLE

#include <distance.h>

...

vec v1 = vec_new_string("6.5 6.2 3.5");
vec v2 = vec_new_string("6.5 6.2 3.5");
/* compute the Euclidean distance between the two vectors */
double distance = vec_distance_norm(v1, v2, 2);
