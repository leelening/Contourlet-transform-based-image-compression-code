Name

it_dequantize_vec -- dequantize a vector

Synopsis

vec it_dequantize_vec( it_quantizer_t *q, ivec v );

DESCRIPTION

This function dequantizes the vector v using the dequantizer q and returns the vector of reconstructed values.
