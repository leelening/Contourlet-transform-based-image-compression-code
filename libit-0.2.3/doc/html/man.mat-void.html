mat_void

Name

mat_void -- Set a matrix to void

Synopsis

#include <it/mat.h>

void mat_void( mat m );

DESCRIPTION

The function mat_void sets a matrix to the void matrix, i.e. a matrix containing no element.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

mat m = mat_new_ones(2,2);  /* Matrix of size 2x2                           */
mat_void( m );              /* Matrix does not contains any element anymore */
