mat_new_inv

Name

mat_new_inv -- computes the inverse of the input matrix

Synopsis

#include <it/linalg.h>

mat mat_new_inv( mat m );

DESCRIPTION

The function mat_new_inv outputs a newly created matrix containing the inverse of matrix m when it exists. Otherwise if returns NULL. See: mat_inv

RETURN VALUE

NULL if input matrix is singular, new inverse matrix otherwise

EXAMPLE

#include <linalg.h>
