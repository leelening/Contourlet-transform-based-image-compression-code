vec_set_length_max

Name

vec_set_length_max -- Modify the amount of memory allocated to a vector

Synopsis

#include <it/vec.h>

void vec_set_length_max( vec v, size_t N );

DESCRIPTION

The vec_set_length_max function modifies the amount of memory allocated for a vector so that the maximum number of allocated elements is equal to N. It is the equivalent of vec_new_alloc for existing vectors. This macro may modify the pointer representing the vector.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new(10);      /* allocate a new vector of size 10      */
vec_set_length_max(v,4);  /* modify the amount of allocated memory */
vec_delete(v);            /* free the vector                       */
