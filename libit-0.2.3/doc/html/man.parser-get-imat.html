parser_get_imat

Name

parser_get_imat -- read a matrix

Synopsis

#include <it/parser.h>

imat parser_get_imat( const parser_t * p, const char * varname );

DESCRIPTION

The function parser_get_imat retrieves a matrix variable of type imat with the parser. The variable varname is the name of the variable to be found.

RETURN VALUE

the matrix read by the parser

EXAMPLE

#include <parser.h>

...

parser_t * p = parser_new (argc, arv, NULL, NULL);
imat im      = parser_get_imat (p, "mymat");
