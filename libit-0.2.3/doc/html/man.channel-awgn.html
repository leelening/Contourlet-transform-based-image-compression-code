channel_awgn

Name

channel_awgn -- AWGN channel simulation

Synopsis

#include <it/channel.h>

vec channel_awgn( vec v, double sigma );

DESCRIPTION

This function simulates the transmission of a vector over an additive white Gaussian noise (AWGN) channel. An independent Gaussian noise of standard deviation sigma is added to the vector v.

RETURN VALUE

The noisy signal

EXAMPLE

#include <channel.h>

...

vec message = vec_new_string("-1 1 1 1 -1 -1 1");

/* send over an AWGN channel of standard deviation 0.5 */ 
vec received = channel_awgn(message, 0.5);
