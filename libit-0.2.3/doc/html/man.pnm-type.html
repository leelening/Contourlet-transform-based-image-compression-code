pnm_type

Name

pnm_type -- Type of a pnm file

Synopsis

#include <it/io.h>

char pnm_type( const char * filename );

DESCRIPTION

This function the PNM type character from a PNM image file specified by filename. The only supported filetype by libit is currently type 5. This function can therefore be used to verify a PNM can be read by libit.

RETURN VALUE

The type of the PNM file: '1' binary image in text format '2' grayscale image in text format '3' rgb image in text format '4' binary image in raw (binary) format '5' grayscale image in raw format -> the only format supported by libit '6' rgb image in raw format

EXAMPLE

#include <io.h>

...

char type = pnm_type("image.pgm");
if(type != '5')
  it_error("Unsupported type %c in %s", type, "image.pgm");
