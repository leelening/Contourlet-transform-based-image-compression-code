cmat_width

Name

cmat_width -- get the width of a matrix

Synopsis

#include <it/mat.h>

int cmat_width( cmat m );

DESCRIPTION

This macro returns the width of the matrix m.

RETURN VALUE

The width

EXAMPLE

#include <mat.h>

...

cmat m = cmat_new(5, 7);
int w  = cmat_width(m); /* w = 7 */
