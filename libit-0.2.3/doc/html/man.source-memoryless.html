source_memoryless

Name

source_memoryless -- generic dicrete source generator

Synopsis

#include <it/source.h>

ivec source_memoryless( size_t size, vec pdf );

DESCRIPTION

This function generates a random discrete vector of length size where each sample is drawn from the distribution pdf. The length of the pdf vector gives the cardinality of the source alphabet, with each element representing the probability of the corresponding symbol. Therefore, the sum of the pdf vector must be equal to one.

RETURN VALUE

EXAMPLE

#include <source.h>

...

vec pdf = vec_new_string("0.5 0.3 0.1 0.1"); /* probability density function */
S = source_memoryless(20, pdf );
