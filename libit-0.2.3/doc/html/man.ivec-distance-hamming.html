ivec_distance_hamming

Name

ivec_distance_hamming -- Hamming distance between two sequences

Synopsis

#include <it/distance.h>

int ivec_distance_hamming( ivec v1, ivec v2 );

DESCRIPTION

The function ivec_distance_hamming returns the Hamming distance between two sequences, i.e. the number of positions such that the component of v1 is not equal to the one of v2. The Hamming distance is only defined between two sequences of the same length. In the proposed implementation, the function returns an integer for sequences of different length. This value is computed assuming that the vector v1 is the reference. If the sequences are not of the same length, there are two cases: either the sequence v2 is shorter than the sequence v1. In that case missing symbols are assumed to be in error. If the sequence v2 is longer, extra symbols are discarded but do not increase the distance. Hence, if v1 is a prefix of v2, then this function returns 0.

RETURN VALUE

The Hamming distance

EXAMPLE

#include <distance.h>

...

ivec v1 = ivec_new_string("0 3 0 1");
ivec v2 = ivec_new_string("0 1 0 2");
/* compute the hamming distance between v1 and v2 */
int distance = ivec_distance_hamming(v1, v2);
