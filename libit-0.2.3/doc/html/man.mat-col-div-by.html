mat_col_div_by

Name

mat_col_div_by -- divide the elements of a column by a given value

Synopsis

#include <it/mat.h>

void mat_col_div_by( mat m, size_t col, double a );

DESCRIPTION

The function mat_col_div_by divides all the elements of the column col by the quantity a.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

imat m = imat_new_set (3,2,2);     /* m = [[3 3]
                                           [3 3]]    */

imat_col_div_by (m, 1, 2);         /* m = [[3 1.5]
                                           [3 1.5]]  */
