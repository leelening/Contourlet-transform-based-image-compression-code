imat_get_submatrix

Name

imat_get_submatrix -- Extract submatrix from input matrix

Synopsis

#include <it/mat.h>

imat imat_get_submatrix( imat m, idx_t r1, idx_t c1, idx_t r2, idx_t c2 );

DESCRIPTION

The function imat_get_submatrix creates a new matrix containing elements of m in range [r1,c1]x[r2,c2].

RETURN VALUE

Desired submatrix from input matrix

#include <mat.h>
