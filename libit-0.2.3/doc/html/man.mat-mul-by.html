mat_mul_by

Name

mat_mul_by -- Performs element-wise matrix multiplication

Synopsis

#include <it/mat.h>

void mat_mul_by( mat m, double a );

DESCRIPTION

The function mat_mul_by multiplies every element of matrix m by value a.

RETURN VALUE

void

EXAMPLE

#include <mat.h>

...

mat m = mat_new (2,2); 

m[0][0] = 1; m[0][1] = 3; 
m[1][0] = 2; m[1][1] = 4; 

mat_mul_by (m,2.); 

/*
m = 
[[ 2   6 ]
 [ 4   8 ]]
*/
