bvec_set_between

Name

bvec_set_between -- set a part of a vector with a value

Synopsis

#include <it/vec.h>

void bvec_set_between( bvec v, size_t i1, size_t i2, byte val );

DESCRIPTION

The bvec_set function set the elements of the byte vector v, whose position is between i1 and i2 (included), to the value val .

RETURN VALUE

pointer to updated vector

EXAMPLE

#include <vec.h>
