mat_ivec_mul

Name

mat_ivec_mul -- return the product of a vector by a matrix

Synopsis

#include <it/mat.h>

vec mat_ivec_mul( mat m, ivec v );

DESCRIPTION

The function mat_ivec_mul returns the product of the matrix m by the vector v

RETURN VALUE

The vector

EXAMPLE

#include <mat.h>

...

mat m = mat_new_ones (3,3);
ivec v = ivec_new_ones (3);
vec r = mat_ivec_mul (m, v);  /* r = [3 3 3] */
