vec_void

Name

vec_void -- set a vector to the void vector

Synopsis

#include <it/vec.h>

void vec_void( vec v );

DESCRIPTION

The function vec_void set the vector length of v to 0. The memory allocated to this vector remains allocated.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

v = vec_new_ones(5);
vec_void(v);         /* Set the vector to []  */
vec_delete(v);       /* free the vector       */
