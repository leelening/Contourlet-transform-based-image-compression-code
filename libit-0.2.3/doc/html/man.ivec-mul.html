ivec_mul

Name

ivec_mul -- component per component multiplication of 2 ivec

Synopsis

#include <it/vec.h>

void ivec_mul( ivec v1, ivec v2 );

DESCRIPTION

The function ivec_mul multiplies each component of the vector v1 by the corresponding component of the vector v2. Note that the vectors v1 and v2 must have the same length.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

v1 = ivec_new_string( "1 0 -2 5" );     * vector [1 0 -2 5] */
v2 = ivec_new_string( "-3 1 1 5" );    /* vector [-3 1 1 5] */
ivec_mul( v1, v2 );                    /* v1 = [-3 0 -2 25] */
ivec_delete(v1);
ivec_delete(v2);
