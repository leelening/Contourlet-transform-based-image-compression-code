cvec_count

Name

cvec_count -- Count an element in a vector

Synopsis

#include <it/vec.h>

int cvec_count( cvec v, cplx a );

DESCRIPTION

The function cvec_count counts the number of times that the complex number a occurs in the vector v.

RETURN VALUE

The number of occurences found

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new_string( "i 3 -1+i 3" );    /* Vector [i 3 -1+i 3] */
int c  = cvec_count( v, 3 );                 /* c = 2               */
