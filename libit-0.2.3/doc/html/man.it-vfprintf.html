it_vfprintf

Name

it_vfprintf -- formatted output to a stream, variable args

Synopsis

#include <it/io.h>

void it_vfprintf( FILE * output, char *fmt, va_list ap );

DESCRIPTION

The function it_vfprintf extends the standard C vfprintf function and provides an output according to the format string. Being an extension of vfprintf, it supports all its types and modifiers. See the it_printf function for details on the extensions.

RETURN VALUE

None

EXAMPLE

#include <io.h>

...

void my_it_printf( char *fmt, ... ) {
  va_list ap;
  va_start(ap, fmt);
  it_vfprintf(stdout, fmt, ap);
  va_end(ap);
}
