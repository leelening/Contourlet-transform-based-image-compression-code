bmat_zeros

Name

bmat_zeros -- Set the elements of a matrix to 0

Synopsis

#include <it/mat.h>

void bmat_zeros( bmat m );

DESCRIPTION

The function mat_zeros sets all the elements of the matrix m to 0. Hence, the size of the matrix m is not modified.

RETURN VALUE

None

EXAMPLE

#include <mat.h>

...

bmat m = bmat_new_ones (2,2); /* m = [[1 1]  */
                                      [1 1]  */

bmat_zeros (m);               /* m = [[0 0]  */
                                      [0 0]  */
