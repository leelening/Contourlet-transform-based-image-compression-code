it_printf

Name

it_printf -- formatted output

Synopsis

#include <it/io.h>

void it_printf( char *fmt, ... );

DESCRIPTION

The function it_printf extends the standard C printf function and provides an output according to the format string. Being an extension printf, it supports all its types and modifiers.

Additional format string conversion specification

The following extra escape characters can be using to print a vector or a matrix. To output the actual character instead, it must be doubled in the format string.

$

The following conversion specifier and modifiers apply to the elements of the vector argument.

#

The following conversion specifier and modifiers apply to the elements of the matrix argument.

Additional conversion specifier

z

The cplx argument is output as "a + b * i" where a and b are the real and imaginary part respectively, output in floating point decimal notation.

v

The double element in a vector is output in floating point decimal notation, with a fixed number of decimals. By default this conversion specifier is equivalent to .3f. It can be changed with the it_set_vec_default_fmt function.

m

The double element in a matrix is output in floating point decimal notation, with a fixed number of decimals. By default this conversion specifier is equivalent to 9.3g. It can be changed with the it_set_mat_default_fmt function.

RETURN VALUE

None

EXAMPLE

#include <io.h>

...

ivec v = ivec_new_ones(10); /* a new vector with elements set to 1 */
it_printf("$d
", v);       /* output the vector */
ivec_delete(v);             /* free the vector */
cmat m = cmat_new_ones(3,4); /* create a new matrix of complex */
it_printf("#z
", m);        /* output the matrix */
cmat_delete(m);              /* free the matrix */
