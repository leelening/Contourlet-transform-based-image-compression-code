itf_differentiate

Name

itf_differentiate -- differentiation operator

Synopsis

#include <it/math.h>

double itf_differentiate( double x, it_function_args(itf_differentiate) *args );

DESCRIPTION

The function itf_differentiate defines the differentiate of an it_function_t. The differentiate of the function is approximated by computing the value of the function at x+delta and x-delta, and computing the difference divided by two delta. In the current implementation, delta is set to 1e-5, leading to an error in O(1e-10). The differentiation operator has the following parameters, passed through the args argument.

Function parameters

function

The function to differentiate.

args

The parameters of the function to differentiate.

RETURN VALUE

The differentiate of the function for a given 'x'

EXAMPLE

#include <math.h>

...

/* declare the differentiation parameters */
it_function_args(itf_differentiate) differentiate_args;

/* differentiate the identity function */
differentiate_args.function = itf_identity;
differentiate_args.args = NULL;

/* compute the value of the differentiate at 1.0 */
itf_differentiate(1.0, &differentiate_args);
