cvec_decr

Name

cvec_decr -- Substract a value to the vector components

Synopsis

#include <it/vec.h>

void cvec_decr( cvec v, cplx a );

DESCRIPTION

The function vec_decr substracts the quantity a from all the components of the vector v.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v  = cvec_new_string("0.3-i i 1");  /* vector [ 0.3-i 1+i i ]    */
cplx c = cplx(1,1);
cvec_decr(v, c);                         /* v = [ -0.7-2i 0 -1 ]      */
cvec_delete(v);
