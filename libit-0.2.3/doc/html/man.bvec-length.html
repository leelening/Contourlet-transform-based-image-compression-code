bvec_length

Name

bvec_length -- get the length of a vector

Synopsis

#include <it/vec.h>

int bvec_length( bvec v );

DESCRIPTION

The function bvec_length returns the length of the vector v.

RETURN VALUE

The length

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_ones(7);   /* Vector [1 1 1 1 1 1 1] */
int l  = bvec_length(v);     /* l = 7                  */
