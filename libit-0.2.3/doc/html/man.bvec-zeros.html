bvec_zeros

Name

bvec_zeros -- set all the elements of a vector to zero

Synopsis

#include <it/vec.h>

void bvec_zeros( bvec );

DESCRIPTION

The function bvec_zeros sets all the elements of an already existing vector to 0.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new( 5 ); /* creates a vector of size 5 (not initialized) */
bvec_zeros( v );        /* the vector is set to [0 0 0 0 0]             */
bvec_delete(v);         /* free the vector                              */
