cvec_1N

Name

cvec_1N -- Set a vector to the values 1, 2, ..., N

Synopsis

#include <it/vec.h>

void cvec_1N( cvec );

DESCRIPTION

The function cvec_1N sets the vector v to [1 2 3... ] so that the vector length remains the same.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

cvec v = cvec_new(5); /* creates the vector (not initialized) */
cvec_1N(v);           /* set v to [1 2 3 4 5]                 */
cvec_delete(v);       /* free the vector                      */
