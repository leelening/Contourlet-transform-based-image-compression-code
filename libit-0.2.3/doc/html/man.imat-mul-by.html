imat_mul_by

Name

imat_mul_by -- Performs element-wise matrix multiplication

Synopsis

#include <it/mat.h>

void imat_mul_by( imat m, int a );

DESCRIPTION

The function imat_mul_by multiplies all the values of the matrix m by the same factor a.

RETURN VALUE

void
