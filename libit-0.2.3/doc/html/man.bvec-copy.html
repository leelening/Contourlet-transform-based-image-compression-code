bvec_copy

Name

bvec_copy -- copy a vector into another

Synopsis

#include <it/vec.h>

void bvec_copy( bvec dest, bvec orig );

DESCRIPTION

This function copies the vector orig into the vector dest. The vectors must have the same length and both must be already allocated.

RETURN VALUE

None
