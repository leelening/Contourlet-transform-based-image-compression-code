imat_clone

Name

imat_clone -- Creates a new matrix with elements of input matrix

Synopsis

#include <it/mat.h>

imat imat_clone( imat m );

DESCRIPTION

The function imat_clone creates a new matrix with same dimensions as m and copies elements of m in the newly created matrix.

RETURN VALUE

Cloned matrix
