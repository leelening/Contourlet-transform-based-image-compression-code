it_warning

Name

it_warning -- print an warning

Synopsis

#include <it/io.h>

void it_warning( char *fmt, ... );

DESCRIPTION

This function prints an error to the standard error output (stderr) preceded by "## IT Warning ##".

RETURN VALUE

None

EXAMPLE

#include <io.h>

...

double v = 3.1415;
it_warning("Unexpected value %f", v);
