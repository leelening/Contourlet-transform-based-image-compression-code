cmat_width_max

Name

cmat_width_max -- Maximum width of a matrix

Synopsis

#include <it/mat.h>

int cmat_width_max( cmat m );

DESCRIPTION

This function returns the maximum width of the matrix m

RETURN VALUE

The maximum width

EXAMPLE

#include <mat.h>

...

cmat m;
...
int width = cmat_width_max(m);
