mat_get_row

Name

mat_get_row -- Get new vector from given row of input matrix

Synopsis

#include <it/mat.h>

vec mat_get_row( mat m, idx_t r );

DESCRIPTION

The function mat_get_row creates a new vector containing the element of row r of matrix m.

RETURN VALUE

The desired row as a new vector

EXAMPLE

#include <mat.h>

...

mat m = mat_new (2,2); 
vec r1; 

m[0][0] = 1; m[0][1] = 3;
m[1][0] = 2; m[1][1] = 4;

r1 = mat_get_row (m,1);     /* r1 = [ 1  3 ] */
