vec_abs

Name

vec_abs -- absolute value for the components of a vector

Synopsis

#include <it/vec.h>

void vec_abs( vec v );

DESCRIPTION

The function vec_abs sets all the components of the vector v to their absolute value.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new_string("1 -2.7 7 0");           /* vector [1 -2.7 7 0] */
vec_abs( v );                                   /* v = [1 2.7 7 0]     */
