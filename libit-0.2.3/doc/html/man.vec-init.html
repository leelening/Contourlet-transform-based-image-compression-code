vec_init

Name

vec_init -- Initialization of a vector from a buffer of double

Synopsis

#include <it/vec.h>

void vec_init( vec v, double * buf, size_t N );

DESCRIPTION

The vec_init function copies the N first elements of the memory area pointed by pointer buf. into vector v. Note that the vector v must alreadly be allocated when this function is called.

RETURN VALUE

None

EXAMPLE

#include <vec.h>

...

vec v = vec_new_zeros(15);/* allocate a new vector of zeros of size 15 */
double * buf = (double *) malloc( 13 * sizeof( double ) );
vec_init(v,buf,13);       /* Copy the start the buffer into the vector */
vec_delete(v);            /* free the vector and the buffer */
free( buf );
