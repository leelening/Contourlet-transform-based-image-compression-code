entropy_bin

Name

entropy_bin -- first-order entropy rate of a binary source

Synopsis

#include <it/source_func.h>

double entropy_bin( double p0 );

DESCRIPTION

This function returns the entropy rate, i.e. the minimum achievable average description length of a binary first-order source described by p0. This value is the probability of the 0 symbol.

RETURN VALUE

EXAMPLE

#include <source_func.h>

...

double p0 = 0.1; /* a binary source with mostly '1' */
double H = entropy_bin(p0);
