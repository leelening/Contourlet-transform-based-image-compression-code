bvec_new_range

Name

bvec_new_range -- create the vector [0, 1, 2, ...N-1]

Synopsis

#include <it/vec.h>

bvec bvec_new_range( size_t N );

DESCRIPTION

The vec_new_range function creates the vector [0 1 ... N-1], where the value N denotes the length of the vector.

RETURN VALUE

The new vector

EXAMPLE

#include <vec.h>

...

bvec v = bvec_new_range(5);  /* creates the vector [0 1 2 3 4] */
bvec_delete(v);              /* free the vector                */
